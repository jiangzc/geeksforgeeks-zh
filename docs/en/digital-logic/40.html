Flip-flop types and their Conversion

Flip-flop is a circuit that maintains a state until directed by input to change the state. A basic flip-flop can be constructed using two-NAND or two-NOR gates.
Types of flip-flops:

  1. RS Flip Flop
  2. JK Flip Flop
  3. D Flip Flop
  4. T Flip Flop
