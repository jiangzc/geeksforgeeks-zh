C程序查找IP地址，子网掩码和默认网关

术语

IP地址 ： IP地址（Internet协议地址的缩写）是一个网络硬件的标识号。 拥有IP地址允许设备通过基于IP的网络（如互联网）与其他设备通信。

子网掩码：子网掩码是一个32位数字，用于通过将IP地址划分为网络地址和主机地址来区分IP地址的网络组件。子网掩码用于设计连接本地​​网络的子网或子网并确定IP地址所属的子网。

默认网关：默认网关用作联网计算机用于将信息发送到另一个网络或Internet中的计算机的接入点或IP路由器。默认情况下，默认情况下默认使用此网关，除非应用程序指定另一个网关。默认网关允许网络上的计算机与另一网络上的计算机进行通信。 没有它，网络与外界隔离。

使用系统命令



要获取IP地址，子网掩码和默认网关，我们在cmd.Here中执行ipconfig命令，我们将使用<stdlib.h>中的system（）在C程序的帮助下执行系统操作： 

#include <stdio.h>
#include <stdlib.h>

int main()
{
  system("c:\\windows\\system32\\ipconfig");
  return 0;
}

使用execl命令

这将显示IPv4地址，子网掩码和默认网关。也可以使用execl（）函数执行相同的操作。为了执行稍后的操作，我们编写代码： 

#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

int main()
{
  execl("c:\\windows\\system32\\ipconfig", "ipconfig", 0);
  return 0;
}
