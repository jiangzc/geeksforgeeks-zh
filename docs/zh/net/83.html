路由器棒| 介绍和配置

先决条件 - 访问和中继端口
交换机通过VLAN（虚拟LAN）划分广播域。 VLAN是来自单个广播域的分区广播域。 交换机不会自己在不同的VLAN上转发数据包。 如果我们想让这些虚拟LAN相互通信，则使用VLAN间路由的概念。

VLAN间路由：

VLAN间路由是指我们使不同的虚拟LAN相互通信而不管VLAN存在的位置（在同一交换机或不同交换机上）的过程。 VLAN间路由可以通过第3层设备实现，即路由器或第3层交换机。 当通过路由器完成VLAN间路由时，它就被称为路由器上的路由器 。

路由器：

路由器的接口分为子接口，作为各自VLAN的默认网关。

配置：



这是一个拓扑，其中有路由器和交换机以及一些终端主机。 在交换机上创建了2个不同的VLAN。 路由器的接口分为2个子接口（因为有2个不同的VLAN），它们将充当各自VLAN的默认网关。 然后路由器将执行VLAN间路由，并且VLAN将相互通信。

首先，我们将IP地址分配给主机PC1为192.168.1.10/24，服务器192.168.1.20/24，另一台主机PC2将手动分配IP地址192.168.2.10/24。

现在，我们将fa0 / 0的子接口设置为fa0 / 0.1和fa0 / 0.2，并在路由器的端口上分别分配IP地址192.168.1.1/24和192.168.2.1/24。 

 r1# int fa0/0.1
r1# encapsulation dot1q 2
r1# ip address 192.168.1.1 255.255.255.0
r1# int fa0/0.2
r1# encapsulation dot1q 3
r1# ip address 192.168.2.1 255.255.255.0

注意：此处封装类型dot1q用于2个不同VLAN之间的帧标记。 当交换机将一个VLAN的数据包转发到另一个VLAN时，它会将VLAN插入以太网报头。

现在，我们将在交换机上创建2个不同的VLAN，即VLAN 2和VLAN 3，其名称为HR_dept和sales_dept 。 

 Switch# vlan 2
Switch# name HR_dept
Switch# vlan 3
Switch# name sales_dept 
 Switch# int range fa0/1-2
Switch# switchport mode access
Switch# switchport access vlan 2
Switch# int fa0/3
Switch# switchport mode access
Switch# switchport access vlan 3

在这里，我们已经将VLAN 2分配给特定的交换机端口fa0 / 1，fa0 / 2和vlan 3到fa0 / 3。

注意：使用 int range fa0 / 1-2命令，因为单个VLAN中存在多个主机。
