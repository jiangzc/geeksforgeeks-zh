LAN Technologies | ETHERNET

局域网（LAN）是连接建筑物或有限地理区域内的各种终端或计算机的数据通信网络。设备之间的连接可以是有线的或无线的。 使用IEEE 802.11的以太网，令牌环和无线LAN是标准LAN技术的示例。

以太网： -
以太网是最广泛使用的LAN技术，它是根据IEEE标准802.3定义的。 其广泛可用性背后的原因是以太网易于理解，实施，维护并允许低成本的网络实施。 此外，以太网在允许的拓扑方面提供了灵活性。 以太网以两层OSI模型运行，即物理层和数据链路层。 对于以太网，协议数据单元是Frame，因为我们主要处理DLL。 为了处理冲突，以太网中使用的访问控制机制是CSMA / CD。
曼彻斯特编码技术用于以太网。

由于我们所讨论的是IEEE 802.3标准以太网，因此0表示为从高到低的转换，1表示从低到高的转换。 在曼彻斯特编码和差分曼彻斯特编码中，Buad速率是比特率的两倍。 

 Baud rate = 2* Bit rate

以太网LAN由网络节点和互连媒体或链路组成。 网络节点可以有两种类型：
数据终端设备（DTE）： -通常，DTE是将用户信息转换为信号或重新转换接收信号的终端设备。 DTE设备是：个人计算机，工作站，文件服务器或打印服务器，也称为终端站。 这些设备是数据帧的源或目标。 数据终端设备可以是单件设备或多件设备，它们互连并执行所有必需的功能以允许用户进行通信。 用户可以与DTE交互，或者DTE可以是用户。

数据通信设备（DCE）： - DCE是通过网络接收和转发帧的中间网络设备。 它们可以是独立设备，例如中继器，网络交换机，路由器，也可以是通信接口单元，例如接口卡和调制解调器。 DCE执行诸如信号转换，编码之类的功能，并且可以是DTE或中间设备的一部分。

目前，这些数据速率是针对光纤和双绞线电缆的操作而定义的：
i）快速以太网
快速以太网是指可以100 Mbit / s的速率传输数据的以太网。



ii）千兆以太网
千兆以太网提供1,000 Mbit / s（1 Gbit / s）的数据速率。

iii）万兆以太网
万兆以太网是最新一代产品，其数据速率为10 Gbit / s（10,000 Mbit / s）。 它通常用于需要高数据速率的高端应用中的主干。

ALOHA
Aloha协议是夏威夷大学项目的一部分。 它提供了几个涉及分组无线电网络的夏威夷群岛的计算机之间的数据传输。 Aloha是数据链路层的多址协议，提出了多个终端如何在没有干扰或冲突的情况下访问媒体。

ALOHA有两种不同的版本：
1.纯阿罗哈
Pure Aloha是一种无插槽，分散且易于实现的协议。 在纯ALOHA中，站点只需在想要发送数据时发送帧。 在发送之前，它不会检查通道是否忙碌。 如果两个或多个站同时发送，则发生冲突并且帧被破坏。 无论何时任何站发送帧，它都期望来自接收器的确认。 如果未在指定时间内收到，则该站假定该帧或确认已被销毁。 然后，电台等待一段随机时间并再次发送帧。 这种随机性有助于避免更多的碰撞。 这种方案适用于负载不大的小型网络。 但是在大量负载的网络中，这种方案很难实现。 这导致了Slotted Aloha的发展。
确保纯粹的aloha：预测其帧的吞吐量和传输速率。
为此做出一些假设：
i）所有帧的长度应相同。
ii）站在发送或尝试发送帧时不能生成帧。
iii）站的数量试图根据泊松分布进行传输（新帧和碰撞的旧帧）。 

 Vulnerable Time = 2 * Tt

纯ALOHA的效率： 

 S pure= G * e^-2G 
where G is number of stations wants to transmit in Tt slot.

Maximum Efficiency:
Maximum Efficiency will be obtained when G=1/2

( S pure) max = 1/2 * e^-1
           = 0.184 

Which means, in Pure ALOHA, only about 18.4% of the time is used for successful transmissions.

2.开槽阿罗哈
这与Pure Aloha非常相似，仅在传输方式上有所不同。 发送方等待一段时间，而不是在需求时发送。 在时隙ALOHA中，共享信道的时间被划分为称为时隙的离散间隔。 这些站仅有资格在时隙的开头发送帧，并且每个时隙只发送一帧。 如果任何电台无法将帧放置在插槽开头的频道上，则必须等到下一个时隙开始。 如果两个站尝试在同一时隙的开头发送，则仍有可能发生冲突。 但是仍然可能发生的碰撞次数大幅减少，与Pure Aloha相比，性能变得更好。

只有当前插槽才可能发生冲突。 因此，脆弱时间是Tt。

开槽ALOHA的效率： 

 S slotted = G * e^-G

Maximum Efficiency:
( S slotted) max = 1 * e^-1 
              = 1/e = 0.368 
Maximum Efficiency, in Slotted ALOHA, is 36.8%.
