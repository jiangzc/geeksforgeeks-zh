电路交换

在电路交换中，网络资源（带宽）被分成多个部分，并且在连接期间比特延迟是恒定的。 在发送器和接收器之间建立的专用路径/电路提供有保证的数据速率。 一旦建立电路，就可以毫无延迟地传输数据。
电话系统网络是电路交换的一个例子。 TDM（时分复用）和FDM（频分复用）是将多个信号复用到单个载波中的两种方法。

  • 频分复用： 分为多个频段
    当多个数据信号被组合以通过共享通信介质同时传输时，使用频分复用或FDM。这是一种将总带宽划分为一系列非重叠频率子带的技术，其中每个子带携带不同的信号。 在无线电频谱和光纤中实际使用以共享多个独立信号。
  • 时分复用： 分为帧
    时分复用（TDM）是一种通过传输线两端的同步开关在公共信号路径上发送和接收独立信号的方法。 TDM用于长距离通信链路，并承受来自最终用户的大量数据流量负载。
    时分复用（TDM）也称为数字电路交换。

电路交换中的公式： 

 Transmission rate = Link Rate or Bit rate / 
                    no. of slots = R/h bps
Transmission time = size of file / 
                    transmission rate 
                 = x / (R/h) = (x*h)/R second
Total time to send packet to destination =
               Transmission time + circuit setup time

示例1：通过电路交换网络从主机A向主机B发送'x位'文件需要多长时间，该网络使用具有'h slot'的TDM并且比特率为'R Mbps'，电路建立时间为k秒。查找总时间？

说明：
传输速率=链路速率或比特率/否。 槽= R / h bps
传输时间=文件大小/传输速率= x /（R / h）=（x * h）/ R.



总时间=传输时间+电路设置时间=（x * h）/ R秒+ k秒


例2：如果链路发送F帧/秒并且每个时隙有B比特，那么找到传输速率？

说明：
由于没有提到每帧中有多少个插槽我们采用一个帧有一个插槽。
传输速率是1秒内发送的数据量。
传输速率= F * B比特/秒
