DHCP中继代理

先决条件 - 动态主机配置协议（DHCP） ， DHCP服务器如何动态地为主机分配IP地址？

为了动态地为主机分配IP地址，DHCP客户端在DORA过程中与DHCP服务器交换DHCP消息。

在DORA过程中，广播发现和请求消息，提供和确认消息是广播或单播，取决于广播标志的值，即如果广播标志的值是1，则广播提供和确认消息，如果为0 ，消息是单播的。 但这仅在DHCP服务器存在于同一网络中时才有效，因为路由器不转发任何广播数据包。 如果服务器存在于不同的网络中会怎样？ 这里有DHCP中继代理的作用。

DHCP中继代理 -
DHCP中继代理是任何TCP / IP主机，用于在服务器存在于不同网络时转发DHCP服务器和客户端之间的请求和回复。 中继代理接收DHCP消息，然后生成新的DHCP消息以在另一个INTERFACE上发送。 此外，DHCP中继代理添加giaddr（数据包的网关地址）字段，如果启用，还添加中继代理信息选项82。 将服务器回复转发给主机时，将删除选项字段。

注 - DHCP中继代理发送和请求消息是单播的。



示例 -

这是一种拓扑，其中有一个没有IP地址的DHCP客户端。 有一个IP地址为192.168.2.2的DHCP服务器，中间有一个路由器，因为DHCP中继代理在接口fa0 / 0上有一个IP地址192.168.1.1，在接口fa0 / 1上有一个192.168.2.1。

现在，首先由DHCP客户端广播发现消息，以找出由swiitch接收的DHCP服务器，因为它在同一个广播域中。交换机在网络中广播dhcp数据包，由PC和路由器接收（ DHCP中继代理）.PC和路由器接收广播数据包但PC丢弃数据包，因为DHCP服务器存在于不同的网络中，因此数据包只能传送到默认网关。

启用DHCP中继代理功能的路由器将源地址替换为自己的地址，使用DHCP服务器IP地址替换目标IP地址，即DHCP中继代理将数据包单播到DHCP服务器。 DHCP中继代理将giaddr字段添加到数据包中并将其转发到DHCP服务器。 将giaddr字段添加到数据包中，以便服务器知道从哪个池中分配IP地址。

在我们的例子中，giaddr字段将包含192.168.1.1（路由器[DHCP中继代理]接收发现消息的接口的IP地址）。

服务器向提供释放的IP地址的路由器回复单播DHCP提议。

作为回报，路由器将DHCP Offer消息广播到已发送DHCP请求的网络。广播消息由交换机接收，如上图所示。

交换机向主机广播DHCP提供消息。 因此，DHCP客户端收到该消息

现在，dhcp客户端广播dhcp请求消息，显示接受交换机接收的服务器的IP地址。交换机将DHCP请求消息广播到其他主机和路由器（DHCP中继代理），如图所示在上图中。

路由器用自己的IP地址替换源IP地址，目的IP保持不变，即dhcp中继代理单播

服务器向路由器（DHCP中继代理）回复单播DHCP确认消息，如上图所示。

路由器依次向网络广播DHCP确认消息，它接收到IP地址的DHCP请求。交换机接收广播消息，如上图所示。

DHCP客户端收到广播DHCP确认消息，因为交换机通过其所有端口广播消息。

配置 -

这是客户端，路由器，dhcp_server的基本配置。路由器已在fa0 / 0上分配IP地址192.168.1.1，在fa0 / 1上分配192.168.2.1。 

 ROUTER(config)#int f0/0
ROUTER(config-if)#ip address 192.168.1.1 255.255.255.0
ROUTER(config-if)#no shutdown
ROUTER(config)#int f0/1
ROUTER(config-if)#ip address 192.168.2.1 255.255.255.0
ROUTER(config-if)#no shutdown

在接口fa0 / 0上为DHCP_SERVER分配IP地址192.168.2.2，并使用名称POOL1定义DHCP池，并将具有子网掩码255.255.255.0的网络192.168.1.0分配给池POOL1。 

 
DHCP_SERVER(config)#int f0/0
DHCP_SERVER(config-if)#ip address 192.168.2.2 255.255.255.0
DHCP_SERVER(config-if)#no shutdown
DHCP_SERVER(config)#ip dhcp pool POOL1
DHCP_SERVER(dhcp-config)#network 192.168.1.0 255.255.255.0
DHCP_SERVER(dhcp-config)#default-router 192.168.1.1
DHCP_SERVER(dhcp-config)#exit

现在，ip helper address命令用于将路由器配置为dhcp中继代理，给出192.168.2.2 DHCP_server的地址。 

 ROUTER(config)#int f0/0
ROUTER(config-if)#ip helper-address 192.168.2.2
ROUTER(config-if)#exit
