基于多精度算术库的RSA算法

公钥加密也称为非对称加密是一种加密类型，它涉及使用两个密钥，即公钥和私钥。 接收方的公共密钥用于由发送方加密明文，而接收方的私钥用于解密加密的消息，因此只能由预期的接收方解密。

RSA（Rivest Shamir Adleman）是一种公钥加密算法，其中密钥生成基于两个大素数p和q的乘积，得到N，即N = p×q。 该算法的安全性的关键在于攻击者需要首先通过在指数时间内发生的N的因子分解来找出p和q。 研究表明，如果N是100位数字，这可能需要70多年。 由于这种复杂性，攻击者无法找到解密密钥d，因为d取决于p，q和加密密钥e。 因此，即使攻击者已知N和e，也无法计算d。

RSA的当前情景

截至2017年4月，RSA-2048在未来许多年内可能无法计算。 此外，最近的勒索软件病毒还使用RSA-2048加密受感染系统上的文件。 如果没有decrption密钥的可用性，则无法解密这些文件，因为无法对这么大的密钥进行分解。 因此，RSA-1024和RSA-2048现在普遍用于安全通信。


上图显示了RSA算法的3个不同阶段。 考虑到素数生成器生成1024位素数p和q，结果N将是2048位数。 由于在加密和解密期间执行的所有模运算都是关于20（即2048位数）的N，因此软件实现可能是耗时的。 此外，C的unsigned long long int数据类型将计算限制为64位数。 为了支持生成大尺寸密钥作为RSA算法的要求和快速计算涉及大尺寸模数的加密和解密，可以使用称为GMP（GNU多精度算术库）的库。 使用此库将允许整个RSA算法在简单的64位操作系统上运行，从而避免使用超级计算机或高配置硬件设备。

什么是GMP？



GMP是一个开源库，允许对有符号整数，有理数和十进制数执行算术计算，除了运行机器的配置外，对其精度没有任何实际限制。 该库在涉及非常大或高精度数的算术计算期间使用，其中大多数用于加密算法。 使用该库的好处是它提供了对任意精度数的支持，其大小在执行程序之前是未知的。 为此库提供的基本接口适用于C语言。 但其他语言包括包装，包括Ada，C ++，C＃，Julia，OCaml，Perl，PHP，Python，R，Ruby和Wolfram语言。

作为示例，下面提到执行两个多个精度数的乘法的C程序。
请注意 ，gmp.h文件需要包含在头文件中。 可以使用该命令在Unix系统上编译这样的程序 

 gcc program_name.c -lgmp

#include <stdio.h>
#include <gmp.h>

int main(void) {
   mpz_t x, y, result;

   mpz_init_set_str(x, "7612058254738945", 10);
   mpz_init_set_str(y, "9263591128439081", 10);
   mpz_init(result);

   mpz_mul(result, x, y);
   gmp_printf("    %Zd\n"
              "*\n"
              "    %Zd\n"
              "--------------------\n"
              "%Zd\n", x, y, result);

   /* free used memory */
   mpz_clear(x);
   mpz_clear(y);
   mpz_clear(result);
  
   return 0;
}

应用

RSA算法已广泛用于许多电子商务应用的安全网络通信和安全交易。 其他应用包括低比特率信道上的语音通信，甚至高速IPSec的安全密钥交换，传送给在线商家时的信用卡详细信息等。
