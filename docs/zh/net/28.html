CSMA / CD的效率

先决条件 - 以太网简介 ，CSMA / CD基础知识

具有冲突检测的载波侦听多路访问（CSMA / CD） -
CSMA方法没有告诉我们在发生碰撞时该怎么做。 具有冲突检测的载波侦听多路访问（CSMA / CD）增加了CSMA算法以处理冲突。 在CSMA / CD中，帧的大小必须足够大，以便发送方在发送帧时可以检测到冲突。 因此，帧传输延迟必须至少是最大传播延迟的两倍 。

假设一些站点传输数据包并成功到达目的地，但它只是最佳情况 ，因此我们必须采用最差情况，其中将存在争用时隙。 争用时隙是由于冲突而无法传输其旅程的那些时隙。 假设一个站发送数据但是碰撞和最坏情况下浪费的时间是2Tp然后一些站B找到了传输数据的方式所以它采取了（如图所示） 

 Tp ( propagation delay) + Tt(transmission time)

现在我们不知道有多少个争用时隙，所以我们认为最坏的情况是n个争用时隙。 

 Efficiency = Tt / ( C*2*Tp + Tt + Tp)  
Tt → transmission time
Tp → propagation time
C → number of collision



在CSMA / CD中，为了成功，只有1个站应该发送而其他站不应该发送。 设p是成功传输数据的概率。 

 P(success) = nC1 * p * (1-p) n-1 (by using Binomial distribution)

对于最大P（成功），区分顶部p并等于零（以获得最大值和最小值）。 

 We get P(max) = 1/e

在获得第一次成功之前我们需要尝试的次数 

 1/P(MAX) = 1/(1/e) = e

这里我们需要尝试的次数（C）= e。

把a = Tt / Tp除以T得效率= Tt /（C * 2 * Tp + Tt + Tp）

我们得到了， 

 Efficiency = 1/(e*2a + 1 + a)
a = Tp/Tt
e = 2.72

Now 
Efficiency = 1/( 1 + 6.44a)

进一步分析效率： 

 Efficiency = 1/ (1 + 6.44a)

           = 1/ {1 + 6.44(Tp/Tt)}

           = 1/ {1 + 6.44((distance/speed)(Bandwidth/packet length))}



从这个推导，我们可以得出很多关系：

  • 如果距离增加，则CSMA的效率降低。
  • CSMA不适合WAN等长途网络; 但最适合LAN。
  • 如果包长度较大，CSMA的效率也会提高; 但最大长度限制为1500字节。
  • 传输时间> = 2 *传播时间
