RC4加密算法

RC4是流密码和可变长度密钥算法。 该算法一次加密一个字节（或一次加密更大的单位）。
密钥输入是伪随机比特生成器，它产生一个8位数的流，在不知道输入密钥的情况下是不可预测的。生成器的输出称为密钥流，一次一个字节与明文流密码组合使用X-OR操作。
例： 

 RC4 Encryption 
10011000 ⊕ 01010000 = 11001000    

RC4 Decryption 
11001000 ⊕ 01010000 = 10011000

密钥生成算法 -
1到256字节的变量 - 英寸键用于初始化256字节状态向量S，其中元素S [0]到S [255]。 对于加密和解密，通过以系统方式选择te 255个条目之一从S生成字节k，然后再次置换S中的条目。

  1. 密钥调度算法：
    初始化 ：S的条目按升序设置为0到255之间的值，创建临时向量T.
    如果密钥k的长度是256字节，那么k被分配给T.否则，对于具有长度（k-len）字节的密钥，从K复制然后K的T的第一个k-len元素被重复为多个填写T的必要时间。这个想法说明如下： 

    for
    i = 0 to 255 do S[i] = i;
T[i] = K[i mod k - len];

    我们使用T来产生S的初始置换。从S [0]到S [255]开始，并且对于每个S [i]算法，根据由T [i]指示的方案将其与S中的另一个字节交换，但是S仍将包含0到255之间的值： 

    j = 0;
for
    i = 0 to 255 do
    {
        j = (j + S[i] + T[i])mod 256;
        Swap(S[i], S[j]);
    }
  2. 伪随机生成算法（Stream Generation）：
    初始化矢量S后，将不使用输入键。 在该步骤中，对于每个S [i]算法，根据由S的当前配置指示的方案将其与S中的另一个字节交换。在达到S [255]之后，该过程继续，再次从S [0]开始。 



    i, j = 0;
while (true)
    i = (i + 1)mod 256;
j = (j + S[i])mod 256;
Swap(S[i], S[j]);
t = (S[i] + S[j])mod 256;
k = S[t];
