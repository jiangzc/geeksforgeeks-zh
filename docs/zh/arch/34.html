管道类型

  • 统一延迟管道
    在这种类型的管道中，所有阶段将花费相同的时间来完成操作。
    在均匀延迟流水线中， 循环时间（Tp）=阶段延迟

    如果阶段之间包含缓冲区，则循环时间（Tp）=阶段延迟+缓冲延迟

  • 非均匀延迟管道
    在这种类型的管道中，不同阶段需要不同的时间来完成操作。
    在这种类型的管道中，循环时间（Tp）=最大（阶段延迟）

    例如，如果有4个具有延迟的级，则为1 ns，2 ns，3 ns和4 ns



    Tp =最大值（1 ns，2 ns，3 ns，4 ns）= 4 ns

    如果阶段之间包含缓冲区，

    Tp =最大值（阶段延迟+缓冲延迟）

    示例：考虑具有级延迟的4段流水线（2 ns，8 ns，3 ns，10 ns）。 找出在上面的管道中执行100个任务所花费的时间。
    解决方案：由于上述管道是非线性管道，
    Tp = max（2,8,3,10）= 10ns
    我们知道ET 流水线 =（k + n - 1）Tp =（4 + 100 - 1）10 ns = 1030 ns

    注意：MIPS =每秒百万条指令

管道与摊位的性能

加速（S）=性能管道 /性能非管道
=> S =平均执行时间非管道 /平均执行时间管道
=> S = CPI 非流水线 *循环时间非流水线 / CPI 流水线 *循环时间流水线

流水线处理器的理想CPI为“1”。 但由于失速，它变得大于'1'。
=> S = CPI 非流水线 *循环时间非流水线 /（1 +每个指令的停顿次数）*循环时间流水线

作为Cycle Time 非管道 = Cycle Time 管道 ，

加速（S）= CPI 非管道 /（1 +每条指令的停顿数）
