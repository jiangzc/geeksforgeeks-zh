几代计算机

介绍：
计算机是操纵信息或数据的电子设备。 它具有存储，检索和处理数据的能力。
如今，计算机可用于键入文档，发送电子邮件，玩游戏和浏览Web。 它还可用于编辑或创建电子表格，演示文稿甚至视频。 但是这个复杂系统的发展始于1946年左右，随着第一代计算机的发展而不断发展。

有五代计算机。

  1. 第一代

    • 介绍：
      1. 1946-1959是第一代计算机时期。
      2. JPEckert和JWMauchy发明了第一台成功的名为ENIAC的电子计算机，ENIAC代表“电子数字集成和计算器”。
    • 几个例子是：
      1. ENIAC
      2. EDVAC
      3. UNIVAC
      4. IBM-701
      5. IBM-650

        6. ...

    • 优点：
      1. 它使用真空管，这是当时唯一可用的电子元件。
      2. 这些计算机可以以毫秒计算。
    • 缺点：
      1. 这些尺寸非常大，重量约为30音。
      2. 这些计算机基于真空管。
      3. 这些计算机非常昂贵。
      4. 由于磁鼓的存在，它只能存储少量信息。
      5. 由于第一代计算机的发明涉及真空管，因此这些计算机的另一个缺点是，真空管需要大的冷却系统。
      6. 工作效率极低。
      7. 有限的编程功能和穿孔卡用于获取输入。
      8. 能耗大。
      9. 不可靠且不断维护。


  2. 第二代

    • 介绍：
      1. 1959-1965是第二代计算机时期。
      2. 第二代计算机基于晶体管而不是真空管。
    • 几个例子是：
      1. 霍尼韦尔400
      2. IBM 7094
      3. CDC 1604
      4. CDC 3600
      5. UNIVAC 1108

        6. … 还有很多

    • 优点：
      1. 由于存在晶体管而不是真空管，电子元件的尺寸减小。 与第一代计算机相比，这导致减小了计算机的尺寸。
      2. 能量减少，产生的热量不如第一代产生的热量。
      3. 汇编语言和穿孔卡用于输入。
      4. 成本低于第一代计算机。
      5. 更快的速度，以微秒为单位计算数据。
      6. 与第一代相比，具有更好的便携性
    • 缺点：
      1. 需要冷却系统。
      2. 需要持续维护。
      3. 仅用于特定目的。


  3. 第三代

    • 介绍：
      1. 1965-1971是第三代计算机时期。
      2. 这些计算机基于集成电路。
      3. IC由Robert Noyce和Jack Kilby于1958-1959发明。
      4. IC是包含多个晶体管的单个元件。
    • 几个例子是：
      1. PDP-8
      2. PDP-11
      3. ICL 2900
      4. IBM 360
      5. IBM 370

        6. … 还有很多

    • 优点：
      1. 与第二代计算机相比，这些计算机更便宜。
      2. 它们快速可靠。
      3. 在计算机中使用IC提供了小尺寸的计算机。
      4. IC不仅减小了计算机的尺寸，而且与以前的计算机相比，它还提高了计算机的性能。
      5. 这一代计算机具有很大的存储容量。
      6. 而不是打卡，鼠标和键盘用于输入。
      7. 他们使用操作系统来实现更好的资源管理，并使用了分时和多编程的概念。
      8. 这些计算机将计算时间从微秒减少到纳秒。
    • 缺点：
      1. IC芯片很难维护。
      2. 制造IC芯片所需的高度复杂的技术。
      3. 需要空调。


  4. 第四代

    • 介绍：
      1. 1971-1980是第四代计算机时期。
      2. 该技术基于微处理器。
      3. 微处理器在计算机中用于在任何程序中执行的任何逻辑和算术功能。
      4. 利用图形用户界面（GUI）技术为用户提供更多的舒适度。
    • 几个例子是：
      1. IBM 4341
      2. DEC 10
      3. STAR 1000
      4. PUP 11

        5. … 还有很多

    • 优点：
      1. 与上一代计算机相比，最快的计算和尺寸减小。
      2. 产生的热量可以忽略不计。
      3. 与上一代计算机相比，尺寸较小。
      4. 减少维护工作量。
      5. 所有类型的高级语言都可以在这种类型的计算机中使用。
    • 缺点：
      1. 微处理器的设计和制造非常复杂。
      2. 由于存在IC，在许多情况下需要空调。
      3. 制造IC需要先进的技术。


  5. 第五代

    • 介绍：
      1. 1980年以后的第五代时期。
      2. 这一代基于人工智能。
      3. 第五代的目的是制造一种能够响应自然语言输入并能够学习和自我组织的设备。
      4. 这一代基于ULSI（超大规模集成）技术，导致生产具有一千万个电子元件的微处理器芯片。
    • 几个例子是：
      1. 桌面
      2. 笔记本电脑
      3. 笔记本
      4. UltraBook的
      5. 的Chromebook

        6. … 还有很多

    • 优点：
      1. 它更可靠，工作更快。
      2. 它有不同的尺寸和独特的功能。
      3. 它为计算机提供了更多用户友好的界面和多媒体功能。
    • 缺点：
      1. 他们需要非常低级的语言。
      2. 他们可能会让人的大脑变得沉闷和注定失败。
