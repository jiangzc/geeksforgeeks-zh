基本计算机指令

基本计算机具有16位指令寄存器（IR），可以表示存储器参考或寄存器参考或输入输出指令。

  1. 内存参考 -这些指令将内存地址称为操作数。 另一个操作数总是累加器。 指定12位地址，3位操作码（除111之外）和1位寻址模式，用于直接和间接寻址。

    示例 -
    IR寄存器包含= 0001XXXXXXXXXXXX，即取指令解码后的ADD，我们发现它是ADD操作的存储器参考指令。 

     Hence, DR <- M[AR]
AC <- AC+ DR, SC <- 0
  2. 寄存器参考 -这些指令对寄存器而不是存储器地址执行操作。 IR（14-12）为111（与存储器参考区分），IR（15）为0（将其与输入/输出指令区分开）。 其余12位指定寄存器操作。

    示例 -
    IR寄存器包含= 0111001000000000，即取指和解码周期后的CMA，我们发现它是补码累加器的寄存器参考指令。 

     Hence, AC <- ~AC
  3. 输入/输出 -这些指令用于计算机和外部环境之间的通信。 IR（14-12）为111（与存储器参考区分），IR（15）为1（将其与寄存器参考指令区分开）。 其余12位指定I / O操作。

    示例 -
    IR寄存器包含= 1111100000000000，即取指和解码周期后的INP，我们发现它是输入字符的输入/输出指令。 因此，来自外围设备的INPUT字符。

包含在16位IR寄存器中的指令集是：

  1. 算术，逻辑和移位指令（和，添加，补充，左，右等循环）
  2. 要在内存中移入信息（存储累加器，加载累加器）
  3. 带状态条件的程序控制指令（分支，跳过）
  4. 输入输出指令（输入字符，输出字符）
符号 十六进制代码 描述
和 0XXX 8XXX 和记忆词到AC
加 1XXX 9XXX 将内存字添加到AC
LDA 2XXX AXXX 将存储器字加载到AC
STA 3XXX Bxxx 将AC内容存储在内存中
包子 4XXX CXXX 无条件分支
BSA 5XXX DXXX 将内存字添加到AC
ISZ 6XXX EXXX 如果为0，则递增并跳过
CLA 7800 清除AC
CLE 7400 清除E（溢出位）
CMA 7200 补充AC
CME 7100 补充E.
CIR 7080 循环右AC和E.
CIL 7040 左侧AC和E循环
INC 7020 增加AC
温泉 7010 如果AC> 0，则跳过下一条指令
SNA 7008 如果AC <0，则跳过下一条指令
SZA 7004 如果AC> 0，则跳过下一条指令
SE 7002 如果E = 0，则跳过下一条指令
HLT 7001 停止电脑
INP F800 输入AC字符
OUT F400 来自AC的输出字符
滑雪 F200 跳过输入标志
SKO F100 跳过输出标志
IEN F080 中断开启
