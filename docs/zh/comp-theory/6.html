正则表达式，常规语法和常规语言

正如乔姆斯基层次结构中所讨论的，常规语言是最受限制的语言类型，并且被有限自动机所接受。

常用表达
正则表达式用于表示常规语言。 表达式是常规的，如果：

  • ɸ是常规语言regular的正则表达式。
  • ɛ是常规语言{ɛ}的正则表达式。
  • 如果a∈Σ（Σ表示输入字母表 ），则a是具有语言{a}的正则表达式。
  • 如果a和b是正则表达式，则a + b也是具有语言{a，b}的正则表达式。
  • 如果a和b是正则表达式，则ab（a和b的连接）也是规则的。
  • 如果a是正则表达式，则a *（0或更多次a）也是常规的。

    如何解决正则表达式和常规语言的问题？

    问题1：正则表达式描述了字母{0,1}中的以下哪种语言？
    （0 + 1）* 0（0 + 1）* 0（0 + 1）*
    （A）包含子串00的所有字符串的集合。
    （B）包含最多两个0的所有字符串的集合。
    （C）包含至少两个0的所有字符串的集合。
    （D）以0或1开头和结尾的所有字符串的集合。

    解决方案：选项A表示它必须具有子字符串00.但是10101也是语言的一部分，但它不包含00作为子字符串。 所以这不是正确的选择。
    选项B表示最多可以有两个0，但00000也是语言的一部分。 所以这不是正确的选择。
    选项C表示它必须包含至少两个0.在正则表达式中，存在两个0。 所以这是正确的选择。
    选项D表示它包含以0或1开头和结尾的所有字符串。但它可以生成以0开头并以1结尾的字符串，反之亦然。 所以这是不正确的。

    问题2：给定语法生成以下哪种语言？
    S - > aS | bS | ε
    （A）{a n b m | n，m≥0}
    （B）{w∈{a，b} * | w具有相同数量的a和b}
    （C）{a n | n≥0}∪{b n | n≥0}∪{a n b n | n≥0}
    （D）{a，b} *

    解决方案：选项（A）表示它将具有0或更多a后跟0或更多b。 但S - > bS => baS => ba也是语言的一部分。 所以（A）不正确。
    选项（B）表示它将具有相同的号码。 a和b的。 但是S - > bS => b也是语言的一部分。 所以（B）不正确。
    选项（C）表示它将具有0或更多a或0或更多b或a后跟b。 但如选项（A）所示，ba也是语言的一部分。 所以（C）不正确。
    选项（D）表示它可以按任何顺序包含任意数量的a和任意数量的b。 所以（D）是正确的。

    问题3：正则表达式0 *（10 *）*表示相同的设置
    （A）（1 * 0）* 1 *
    （B）0 +（0 + 10）*
    （C）（0 + 1）* 10（0 + 1）*
    （D）这些都不是

    解决方案：如果它们生成的语言相同，则两个正则表达式是等效的。
    选项（A）可以生成101但是0 *（10 *）*不能生成。 所以他们不等同。
    选项（B）可以生成0100但是0 *（10 *）*不能生成。 所以他们不等同。
    选项（C）将10作为子字符串，但0 *（10 *）*可能会或可能不会。 所以他们不等同。
