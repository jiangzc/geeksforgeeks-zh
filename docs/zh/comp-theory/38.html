递归和递归可枚举语言

递归可枚举（RE）或类型-0语言

RE语言或0型语言由0型语法生成。 图灵机可以接受或识别RE语言，这意味着它将进入语言串的最终状态，并且对于不属于该语言的字符串可能会或可能不会进入拒绝状态。 这意味着TM可以永久循环不是语言的一部分的字符串。 RE语言也称为图灵可识别语言。

递归语言（REC）

递归语言（RE的子集）可以由图灵机决定，这意味着它将进入语言串的最终状态，并拒绝不属于语言的字符串的状态。 例如; L = {a n b n c n | n> = 1}是递归的，因为我们可以构造一个图灵机，如果字符串的形式是anbncn，则它将移动到最终状态，否则会移动到非最终状态。 因此TM在这种情况下总是会停止。 REC语言也称为图灵可判定语言。 RE和REC语言之间的关系如图1所示。

GATE问题

问题1：以下哪项陈述是假的？
对于每个非确定性TM，存在等价的确定性TM。
2.确认语言是在联合和补充下关闭的。
3.Tint可判定语言在交叉和互补下关闭。
4.识别语言在联合和交集下是封闭的。

A.1和4
B.1和3
C.2
D.3

解：

声明1是正确的，因为我们可以将每个非确定性TM转换为确定性TM。
声明2是错误的，因为图灵可识别语言（RE语言）不会在补充下关闭。
声明3是正确的，因为图灵可判定语言（REC语言）在交叉和互补下是封闭的。
声明4是正确的，因为图灵可识别语言（RE语言）在联合和交集下是封闭的。

问题2：设L为语言，L'为其补语。 以下哪一项不可行？
A. L和L'都不是RE。
B.L和L'中的一个是RE，但不是递归的; 另一个不是RE。
C.L和L'都是RE但不是递归的。
D.L和L'都是递归的。

解：

选项A是正确的，因为如果L不是RE，则其互补将不是RE。 选项B是正确的，因为如果L是RE，则L'不需要是RE，反之亦然，因为RE语言在互补下不会关闭。
选项C是假的，因为如果L是RE，则L'将不是RE。 但是如果L是递归的，L'也将是递归的，并且两者也将是RE，因为REC语言是RE的子集。 正如他们提到的那样不是REC，所以选项是错误的。
选项D是正确的，因为如果L是递归的，L'也将是递归的。

问题3：让L1成为递归语言，让L2成为递归可枚举但不是递归语言。 以下哪一项是正确的？

A.L1'是递归的，L2'是递归可枚举的
B.L1'是递归的，L2'不是递归可枚举的
C.L1'和L2'是递归可枚举的
D.L1'是递归可枚举的，L2'是递归的
解：

选项A是假的，因为L2'不能递归枚举（L2是RE，RE在补充下不关闭）。
选项B是正确的，因为L1'是REC（REC语言在补充下关闭）而L2'不是递归可枚举的（RE语言在补充下不关闭）。
选项C是假的，因为L2'不能递归枚举（L2是RE，RE在补充下不关闭）。
选项D是假的，因为L2'不能递归枚举（L2是RE，RE语言在补充下不会关闭）。 由于REC语言是RE的子集，因此L2'也不能是REC。
