将无上下文语法转换为乔姆斯基范式

先决条件 - 简化上下文无关语法

如果所有生产规则满足以下条件之一，则无上下文语法（CGF）采用乔姆斯基范式（CNF）：

  • 非终端生成终端（例如; X-> x）
  • 非终端产生两个非终端（例如：X-> YZ）
  • 开始符号生成ε。 （例如; S->ε）

考虑以下语法， 

 G1 = {S->a, S->AZ, A->a, Z->z} 
G2 = {S->a, S->aZ, Z->a}

语法G1在CNF中，因为生产规则满足为CNF指定的规则。 然而，语法G2不在CNF中，因为生成规则S-> aZ包含终端，其后是非终端，其不满足为CNF指定的规则。

注意 -



  • 对于给定的语法，可以有多个CNF。
  • CNF生成与CFG生成的语言相同的语言。
  • CNF被用作许多CFG算法的预处理步骤，如CYK，自下而上的解析器等。

如何将CFG转换为CNF？

步骤1.消除RHS的起始符号。
如果起始符号S位于语法中任何生产的RHS，则创建一个新生产：
S0->取值
其中S0是新的起始符号。

第2步。消除空，单位和无用的制作。
如果CFG包含null，unit或无用的生产规则，则将其删除。 您可以参考本文来消除这些类型的生产规则 。

步骤3.如果与其他终端或非终端存在终端，则从RHS消除终端。 例如，; 生产规则X-> xY可以分解为：
X-> ZY
Z-> X

步骤4.使用两个以上的非终端消除RHS。
例如，; 生产规则X-> XYZ可以分解为：
X-> PZ
P-> XY

示例 -我们举一个例子将CFG转换为CNF。 考虑给定的语法G1： 

 S → ASB
A → aAS|a|ε 
B → SbS|A|bb

步骤1.当起始符号S出现在RHS上时，我们将创建一个新的生产规则S0-> S. 因此，语法将成为： 

 S0->S
S → ASB
A → aAS|a|ε 
B → SbS|A|bb

步骤2.由于语法包含空产生A->ε，因此从语法中删除它会产生： 

 S0->S
S → ASB|SB
A → aAS|aS|a 
B → SbS| A|ε|bb

现在，它创建了空产B→ε，它从语法中删除产生： 

 S0->S
S → AS|ASB| SB| S
A → aAS|aS|a 
B → SbS| A|bb

现在，它创建单位生产B-> A，从语法中删除它： 

 S0->S
S → AS|ASB| SB| S
A → aAS|aS|a 
B → SbS|bb|aAS|aS|a

此外，从语法中删除单位生产S0-> S得出： 

 S0-> AS|ASB| SB| S
S → AS|ASB| SB| S
A → aAS|aS|a 
B → SbS|bb|aAS|aS|a

此外，从语法中删除单位生产S-> S得到： 

 S0-> AS|ASB| SB| S
S → AS|ASB| SB
A → aAS|aS|a 
B → SbS|bb|aAS|aS|a

步骤3.在生产规则A-> aAS | aS和B-> SbS | aAS | aS中，端子a和b存在于具有非终止的RHS上。 从RHS中删除它们： 

 S0-> AS|ASB| SB| S
S → AS|ASB| SB
A → XAS|XS|a
B → SYS|bb|XAS|XS|a
X →a
Y→b

此外，B-> bb不能成为CNF的一部分，将其从语法产量中删除： 

 S0-> AS|ASB| SB| S
S → AS|ASB| SB
A → XAS|XS|a
B → SYS|VV|XAS|XS|a
X → a
Y → b
V → b

步骤4：在生产规则S0-> ASB中，RHS有两个以上的符号，将其从语法产量中删除： 

 S0-> AS|PB| SB| S
S → AS|ASB| SB
A → XAS|XS|a
B → SYS|VV|XAS|XS|a
X → a
Y → b
V → b
P → AS

类似地，S-> ASB有两个以上的符号，从语法产量中删除它： 

 S0-> AS|PB| SB| S
S → AS|QB| SB
A → XAS|XS|a
B → SYS|VV|XAS|XS|a
X → a
Y → b
V → b
P → AS
Q → AS

类似地，A-> XAS有两个以上的符号，从语法产量中删除它： 

 S0-> AS|PB| SB| S
S → AS|QB| SB
A → RS|XS|a
B → SYS|VV|XAS|XS|a
X → a
Y → b
V → b
P → AS
Q → AS
R → XA

类似地，B-> SYS有两个以上的符号，从语法产量中删除它： 

 S0 -> AS|PB| SB| S
S → AS|QB| SB
A → RS|XS|a
B → TS|VV|XAS|XS|a
X → a
Y → b
V → b
P → AS
Q → AS
R → XA
T → SY

类似地，B-> XAX有两个以上的符号，从语法产量中删除它： 

 S0-> AS|PB| SB|S
S → AS|QB| SB
A → RS|XS|a
B → TS|VV|US|XS|a
X → a
Y → b
V → b
P → AS
Q → AS
R → XA
T → SY
U → XA

所以这是给定语法所需的CNF。
