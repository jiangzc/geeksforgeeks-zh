上下文敏感语法（CSG）和语言（CSL）

语境敏感语法 -
上下文敏感的语法是一种无限制的语法，其中所有的作品都是形式的 -

其中α和β是非终端和终端的串。

上下文敏感语法比无上下文语法更强大，因为有些语言可以由CSG描述，但不能由无上下文语法描述，而CSL的功能不如无限制语法。 这就是为什么上下文敏感的语法在Chomsky层次结构中位于无上下文和不受限制的语法之间的原因。

上下文敏感语法有4元组。 G = {N，Σ，P，S} ，其中
N =非终端符号集
Σ=终端符号集
S =生产的起始符号
P =有限的制作集
P中的所有规则都是α1Aα2 - >α1βα2的形式

上下文敏感语言：可由上下文敏感语法定义的语言称为CSL。 CSL的属性是：

  • 两个上下文相关语言的联合，交集和串联是上下文敏感的。
  • 上下文相关语言的补充是上下文敏感的。

示例 -



考虑以下CSG。
S→abc / aAbc
Ab→bA
Ac→Bbcc
bB→Bb
aB→aa / aaA
这个语法产生的语言是什么？

方案 ：
S→aAbc
→abAc
→abBbcc
→aBbbcc
→aaAbbcc
→aabAbcc
→aabbAcc
→aabbBbccc
→aabBbbccc
→aaBbbbccc
→aaabbbccc
这个语法生成的语言是{ a n b n c n | N≥1}。
