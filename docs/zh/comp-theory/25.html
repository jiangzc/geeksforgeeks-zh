抽气引理

有两个Pumping Lemmas，定义为
1.常规语言，和
2.上下文 - 自由语言

常规语言的抽象引理
对于任何常规语言L，存在整数n，使得对于所有x∈L与| x | ≥n，存在u，v，w∈Σ*，使得x = uvw，和
（1）| uv | ≤n
（2）| v | ≥1
（3）对于所有i≥0：uv iw∈L

简单来说，这意味着如果字符串v被“抽”，即如果v被插入任意次，则结果字符串仍保留在L中。

Pumping Lemma用于证明语言的不规则性。 因此，如果一种语言是规则的，它总是满足抽水引理。 如果存在至少一个由泵浦制成的不在L中的弦，则L肯定不规则。
与此相反的情况可能并非总是如此。 也就是说，如果Pumping Lemma成立，那并不意味着语言是规则的。

例如，让我们证明L 01 = {0 n 1 n | n≥0}是不规则的。

现在，让x∈L和| x | ≥n。 因此，通过抽水引理，存在u，v，w，使得（1） - （3）成立。

我们表明，对于所有你，v，w，（1） - （3）不成立。
如果（1）和（2）保持，那么x = 0 n 1 n = uvw with | uv | ≤n且| v | ≥1。
所以，u = 0 a ，v = 0 b ，w = 0 c 1 n其中：a +b≤n，b≥1，c≥0，a + b + c = n

uv 0 w = uw = 0 a 0 c 1 n = 0 a + c 1n∉L，因为a + c≠n。



无上下文语言的抽象引理（CFL）
CFL的抽取引理表明，对于任何上下文无关语言L，可以找到两个可以被“抽”的子串，并且仍然使用相同的语言。 对于任何语言L，我们将其字符串分成五个部分并且抽取第二个和第四个子字符串。
此外，Pumping Lemma也被用作证明语言不是CFL的工具。 因为，如果任何一个字符串不满足其条件，则该语言不是CFL。
因此，如果L是CFL，则存在整数n，使得对于所有x∈L与| x | ≥n，存在u，v，w，x，y∈Σ*，这样x = uvwxy，
（1）| vwx | ≤n
（2）| vx | ≥1
（3）对于所有i≥0：uv i wx iy∈L

对于上面的例子，0 n 1 n是CFL，因为任何字符串都可以是在两个地方抽水的结果，一个用于0，另一个用于1。
让我们证明，L 012 = {0 n 1 n 2 n | n≥0}不是无上下文的。
