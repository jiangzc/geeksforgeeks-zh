字符串的DFA至少有两个0和至少两个1

问题 -绘制具有至少两个0和至少两个1的字符串的判断有限自动机（DFA）。

在阅读了这个问题之后，我们首先想到的是我们计算1和0的数量。 此后，如果它们都至少为2，则接受字符串，否则不被接受。 但是我们在DFA中没有任何内存概念，所以我们不能通过这种方法来实现。 

 Input : 1 0 1 1 0 0
Output : Accepted

Input : 1 1 1 0 1 
Output : Not accepted

使用的方法 -
我们观察到的第一件事是0和1都应至少为2.如果其中任何一个小于2，则不接受字符串。 在这个字符串中只接受最后一种情况，其中0和1都至少为2。

州 数为0 数量为1
Q0 0 0
Q1 0 1
Q2 0 > = 2
Q3 1 0
Q4 1 1
Q5 1 > = 2
Q6 > = 2 0
Q7 > = 2 1
Q8接受 > = 2 > = 2



最初0和1的计数均为零，我们处于状态Q0。

  • 步骤1：如果输入为1，则计数1增加到1.转到状态Q1
    如果输入为0，则计数0增加到1.转到状态Q3
  • 步骤2：如果输入为1，则计数1增加到2.转到状态Q2
    如果输入为0，则计数0增加到1.转到状态Q4

  • 步骤3：如果输入为1，则计数1保持增加1.保持相同状态
    如果输入为0，则计数0增加到1.转到状态Q5

  • 步骤4：如果输入为1，则计数1增加到1.转到状态Q4
    如果输入为0，则计数0增加到2.转到状态Q6

  • 步骤5：如果输入为1，则计数1增加到2.转到状态Q5
    如果输入为0，则计数0增加到2.转到状态Q7

  • 步骤6：如果输入为1，则计数1保持增加1.保持相同状态。
    如果输入为0，则计数0增加到2.转到状态Q8

  • 步骤7：如果输入为1，则计数1增加到1.转到状态Q7
    如果输入为0，则计数0保持增加1.保持相同状态。

  • 步骤8：如果输入为1，则计数1增加到2.转到状态Q8
    如果输入为0，则计数0保持增加1.保持相同状态。

  • 步骤9：如果输入为1，则计数1继续增加1.保持相同状态。
    如果输入为0，则计数0保持增加1.保持相同状态。
    如果字符串已完成，则接受
