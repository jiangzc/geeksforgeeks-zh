语法和语言之间的关系

语法是一组生成规则，用于生成语言的字符串。 在本文中，我们讨论了如何查找语法生成的语言，反之亦然。

语法生成的语言 -

给定语法G，其对应的语言L（G）表示从G生成的所有字符串的集合。考虑以下语法， 

 G: S-> aSb|ε

在这个语法中，使用S->ε，我们可以生成ε。 因此，ε是L（G）的一部分。 类似地，使用S => aSb => ab，生成ab。 类似地，也可以生成aabb。
因此， 

 L(G) = {a^nb^n, n>=0}

在上面讨论的语言L（G）中，条件n = 0用于接受ε。
关键点 -

  • 对于给定的语法G，其对应的语言L（G）是唯一的。
  • 对应于语法G的语言L（G）必须包含可以从G生成的所有字符串。
  • 对应于语法G的语言L（G）不得包含任何不能从G生成的字符串。

让我们基于此讨论问题：



阙-1。 考虑语法：（GATE-CS-2009） 

 S -> aSa|bSb|a|b

上述语法在字母{a，b}上生成的语言是以下集合：
（A）所有回文
（B）所有奇数长度的回文。
（C）以相同符号开头和结尾的字符串
（D）所有均匀长度的回文

解决方案：使用S-> a和S-> b，可以生成a和b。 类似地，使用S => aSa => aba，可以生成aba。 可以从语法生成的其他字符串是：a，b，aba，bab，aaa，bbb，ababa，...
因此，选项（B）是正确的。

阙-2。 考虑以下无上下文语法：（GATE-CS-2016）

G1和G2分别生成以下哪一种语言？

解决方案：考虑语法G1：
使用S => B => b，可以生成b。
使用S => B => bB，可以生成bb。
使用S => aS => aB => ab可以生成。
使用S => aS => aB => abB => abb可以生成。
我们可以看到，a的数量可以是零或更多，但b的数量总是大于零。
因此， 

 L(G1) = {a^mb^n| m>=0 and n>0}

考虑语法G2：
使用S => aA => a，可以生成a。
使用S => bB => b，可以生成b。
使用S => aA => aaA => aa可以生成。
使用S => bB => bbB => bb可以生成。
使用S => aA => aB => abB => abb可以生成。

我们可以看到，a或b必须大于0。
因此， 

 L(G2) = {a^mb^n| m>0 or n>0}

语法生成给定语言 -

给定语言L（G），其对应的语法G表示产生L（G）的生成规则。 考虑语言L（G）： 

 L(G) = {a^nb^n, n>=0}

语言L（G）是字符串ε，ab，aabb，aaabbb ......的集合。
对于L（G）中的ε字符串，生成规则可以是S->ε。
对于L（G）中的其他字符串，生产规则可以是S-> aSb | ε。
因此，对应于L（G）的语法G是： 

 S->aSb| ε

关键点 -

  • 对于给定的语言L（G），可以存在多于一个可以产生L（G）的语法。
  • 对应于语言L（G）的语法G必须生成所有可能的L（G）串。
  • 对应于语言L（G）的语法G不得生成任何不属于L（G）的字符串。

让我们基于此讨论问题：

阙-3。 以下哪一个语法生成语言L = {a ^ ib ^ j | 我≠j时}？ （GATE-CS-2006）

解决方案：给定的语言L包含字符串： 

 {a, b, aa, bb, aaa, bbb, aab, abb…}

这意味着字符串必须包含一个或多个OR的数量，一个或多个b OR a，后跟b具有不等数量的a和b。

如果我们在选项（A）中考虑语法，它可以生成ab as： 

 S=>AC=>aAC=>aC=>ab

但是，语言L不能生成ab。因此，选项（A）中的语法不正确。

同样，选项（B）中的语法可以生成ab as： 

 S=>aS=>ab

但是，语言L不能生成ab。因此，选项（B）中的语法不正确。

同样，选项（C）中的语法可以生成ab as： 

 S=>AC=>C=>aCb=>ab

但是，语言L不能生成ab。因此，选项（C）中的语法不正确。
因此，使用消除方法，选项（D）是正确的。
