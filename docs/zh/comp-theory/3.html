乔姆斯基层次结构

根据chomsky层次结构，语法分为4种类型： 

 Type 0 known as unrestricted grammar.
Type 1 known as context sensitive grammar. 
Type 2 known as context free grammar.
Type 3 Regular Grammar.

Type 0（Unrestricted Grammar）

在类型0中
0型语法包括所有正式语法。 类型0语法语言由图灵机识别。 这些语言也称为递归可枚举语言。



语法生成的形式

哪里

是（V + T）* V（V + T）*
V：变量
T：终端。

是（V + T）*。
在类型0中，生产的左侧必须至少有一个变量。

例如，

Sab - > ba
A - > S.

这里，变量是S，A和终端a，b。

类型1（上下文敏感）
Type-1语法生成上下文相关语言。 语法生成的语言由线性绑定自动机识别
在类型1中
1.首先，Type 1语法应该是Type 0。
2.语法生成的形式

| | <= | |

即中的符号计数 小于或等于

例如，
S - > AB
AB - > abc
B - > b

类型2（无上下文）
Type-2语法生成无上下文语言。 语法生成的语言由非确定性下推自动机识别。 Type-2语法生成无上下文语言。
在类型2中，
1.首先它应该是类型1。
2.生产的左侧只能有一个变量。

| | = 1。

他们没有限制 。

例如，
S - > AB
A - > a
B - > b

3型（普通语法）
Type-3语法生成常规语言。这些语言正是可由有限状态自动机决定的所有语言。

类型3是最受限制的语法形式。
类型3应仅为给定形式：

V - > VT * / T *。
（要么）
V - > T * V / T *

例如 ：
S - > ab。
