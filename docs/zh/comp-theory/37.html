构造语言L = {ww |的图灵机 w∈{0,1}}

先决条件 - 图灵机
语言L = {ww | w∈{0,1}}告诉每个0和1的字符串本身都属于这种语言。 解决这个问题的逻辑可以分为两部分：

  1. 找到字符串的中点
  2. 在我们找到中点后，我们匹配符号

示例 -让我们借助一个示例来理解它。 让字符串1 0 1 1 0 1，所以w = 1 0 1并且字符串是形式（ww）。
我们要做的第一件事就是找到中点。 对于他我们将开头的1转换为Y并向右移动直到字符串的结尾。 这里我们将1转换为y。

现在我们的字符串看起来像Y 0 1 1 0 Y.现在向左移动，找到一个X或Y.当我们这样做时，将其右边的0或1分别转换为X或Y，然后在右端执行相同的操作。 现在我们的字符串看起来像YX 1 1 X Y.此后，转换这些1也是最后它看起来像YXY Y X Y.

在这一点上，你已经实现了第一个目标，那就是找到中间点。 现在将中点左侧的所有X和Y分别转换为0和1，因此字符串变为1 0 1 YX Y.现在，将1转换为Y并向右移动直到，在字符串右边部分的开头找到Y将此Y转换为空白（由B表示）。 现在字符串看起来像Y 0 1 BX Y.

类似地，将此应用于0和x后跟1和Y.此字符串看起来像YXYBB B.现在您没有0和1并且字符串右侧的所有X和Y都被转换为空格，因此我们的字符串将被接受。



Assuption：我们将X替换为0，Y替换为1。

使用的方法 -
首先要找到字符串的中点，将字符串开头的0或1分别转换为X或Y，并将相应的0或1转换为字符串末尾的X或Y. 连续完成后，当所有0和1分别转换为X和Y时，达到一个点。 此时你正处于弦乐的中点。 所以，我们首先客观地实现了。

现在，将中点左侧的所有X和Y转换为0和1。 此时，字符串的前半部分是0和1的形式。字符串的后半部分是X和Y的形式。

现在，从字符串的开头开始。 如果你有一个0然后将其转换为X并向右移动直到到达下半部分，如果我们找到X然后将其转换为空白（B）。 然后向后遍历直到找到X或Y.我们将其右边的0或1分别转换为X或Y，相应地，将其后半部分的X或Y转换为空白（B）。

继续这样做，将字符串左边的所有符号转换为X和Y，将字符串右边的所有符号转换为空格。 如果任何一个部分被完全转换但是另一半中的某些符号保持不变，则不接受字符串。 如果你在后半部分没有找到X或Y，则在前半部分分别找到相应的0或1。 然后也不接受字符串。

例子： 

 Input : 1 1 0 0 1 1 0 0 
Output : Accepted

Input : 1 0 1 1 1 0 1
Output : Not accepted
