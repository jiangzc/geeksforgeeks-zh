上下文自由语言的闭包属性

上下文自由语言被下推自动机接受，但不被有限自动机接受。 上下文无关语言可以通过上下文无关语法生成，其形式如下：

A - >ρ（其中A∈N和ρ∈（T∪N）*，N是非终端，T是终端）

上下文自由语言的属性
联盟：如果L1和如果L2是两种无上下文语言，它们的并集L1∪L2也将是无上下文的。 例如，
L1 = {a n b n c m | m> = 0且n> = 0}且L2 = {a n b m c m | n> = 0且m> = 0}
L3 =L1∪L2= {a n b n cm∪an b m c m | n> = 0，m> = 0}也是无上下文的。
L1表示a的数量应该等于b的数量，L2表示b的数量应该等于c的数量。 他们的联盟表明两个条件中的任何一个都是真的。 所以它也是无上下文的语言。

连接：如果L1和If L2是两种无上下文语言，则它们的串联L1.L2也将是无上下文的。 例如，
L1 = {a n b n | n> = 0}，L2 = {c m d m | m> = 0}
L3 = L1.L2 = {a n b n c m d m | m> = 0且n> = 0}也是无上下文的。
L1表示a的数量应该等于b的数量，L2表示c的数量应该等于d's的数量。 他们的串联说首先a的数量应该等于b的数量，那么c的数量应该等于d的数量。 因此，我们可以创建一个PDA，它将首先推动一个's，pop为b'，然后推送c，然后为d进行流行。 所以它可以被下推自动机接受，因此无上下文。

Kleene闭包：如果L1无上下文，其Kleene闭包L1 *也将无上下文。 例如，
L1 = {a n b n | n> = 0}
L1 * = {a n b n | n> = 0} *也是无上下文的。

交集和互补：如果L1和If L2是两种无上下文语言，则它们的交集L1∩L2不需要无上下文。 例如，
L1 = {a n b n c m | n> = 0且m> = 0}且L2 =（a m b n c n | n> = 0且m> = 0}
L3 =L1∩L2= {a n b n c n | n> = 0}不需要上下文。
L1表示a的数量应该等于b的数量，L2表示b的数量应该等于c的数量。 他们的交集说两个条件都必须是真的，但是下推自动机只能比较两个。 所以它不能被下推自动机接受，因此不能没有上下文。
类似地，作为Σ* - L1的无上下文语言L1的互补不需要是无上下文的。

确定性的无语境语言
确定性CFL是CFL的子集，可以通过确定性PDA识别。 确定性PDA只有一个来自给定状态和输入符号的移动。 例如，L1 = {a n b n c m | m> = 0且n> = 0}是DCFL，因为对于a，我们可以推进堆栈，对于b，我们可以弹出。 它可以被确定性PDA识别。 另一方面，L3 = {a n b n cm∪an b m c m | DPDA无法识别n> = 0，m> = 0}，因为a和b的数量可以相等，或者b和c的数量可以相等。 因此，它只能由NPDA实施。 因此，它是CFL但不是DCFL。
注意：出于联合，连接，互补，交叉和kleene闭包，DCFL仅在互补时关闭。

问题：语言L = {0 i 12 i | i≥0}超过字母{0,1,2}是：
A.不递归
B.是递归和确定性的CFL

解决方案：上面的语言是确定性CFL，对于0，我们可以在堆栈上推0，对于2，我们可以弹出相应的0。 由于没有任何歧义可以采取，它是确定性的。 所以，正确的选项是（B）。 由于CFL是递归的子集，因此它也是递归的。
