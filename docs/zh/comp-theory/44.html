有限自动机： 用于识别特定类型输入的模式。 它是最受限制的自动机类型，只能接受常规语言（可以使用OR（+），连接（。），Kleene Closure（*）表示的语言，如a * b *，（a + b）等等。）



确定性FA和非确定性FA： 在确定性FA中，在每个输入符号上每个状态只有一个移动，但在非确定性FA中，对于输入符号，从一个状态可以有零个或多个移动。

注意：

  • NDFA和DFA接受的语言相同。
  • NDFA和DFA的功效相同。
  • NDFA中的州数量小于或等于否。 等效DFA中的州。
  • 对于具有n状态的NFA，在最坏的情况下，DFA中可能的最大状态是2 n
  • 每个NFA都可以转换为相应的DFA。

推送自动机： 下推式自动机具有称为堆栈的额外内存，其提供比有限自动机更多的功率。 它用于识别无上下文语言。

确定性和非确定性PDA： 在确定性PDA中，每个输入符号上的每个状态只有一个移动，但在非确定性PDA中，输入符号的一个状态可以有多个移动。

注意：

  • NPDA的权力超过DPDA。
  • 无法将每个NPDA转换为相应的DPDA。
  • DPDA接受的语言是NPDA接受的语言的子集。
  • DPDA接受的语言称为DCFL（确定性上下文自由语言），它是NPDA接受的NCFL（非确定性CFL）的子集。

线性束缚自动机： 线性束缚自动机具有有限的内存量，称为带，可用于识别上下文敏感语言。

  • LBA比下推自动机更强大。 
    •  FA < PDA < LBA < TM

    图灵机： 图灵机具有无限大小的磁带，用于接受递归可枚待语言。

  • 图灵机可以双向移动。 此外，它不接受ε。
  • 如果字符串不是以语言插入，则机器将在非最终状态下暂停。

    • 确定性和非确定性图灵机： 在确定性图灵机中，每个输入符号上的每个状态只有一个移动，但在非确定性图灵机中，输入符号的一个状态可以有多个移动。

    注意：

    • NTM接受的语言，多磁带TM和DTM是相同的。
    • NTM，Multi-Tape TM和DTM的功能相同。
    • 每个NTM都可以转换为相应的DTM。

    可判定和不可判定的问题：

    如果可以构造图灵机，接受作为语言一部分的字符串并拒绝其他语言，则 语言是可 判定的或递归的 。 例如; 一个数字是素数还是不是一个可判定的问题。

    如果可以构造一个图灵机，它接受作为语言一部分的字符串，并且它可以永久地循环用于不属于语言的字符串，那么 语言就是 半可 判定的或递归的可枚举的 。

    如果我们不能构建一个能给出是或否答案的算法和图灵机，那么 问题是 不可判定的 。 例如; CFG是否含糊不清是不可判定的。



    可数性：

  • 任何有限字母表上的所有字符串集都是可数的。
  • 可数集的每个子集都是有限的或可数的。
  • 所有图灵机都是可数的。
  • 所有不可递归的语言集都是Uncountable。
