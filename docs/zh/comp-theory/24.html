将无上下文语法转换为Greibach范式

先决条件 - 上下文无关语法 ， 简化上下文无关语法

如果所有生产规则满足以下条件之一，则无上下文语法（CGF）采用Geribach Normal Form（GNF）：

  • 非终端生成终端（例如; X-> x）
  • 非终端生成终端，后跟任意数量的非终端（例如：X-> X1X2 ... Xn）
  • 开始符号生成ε。 （例如; S->ε）

考虑以下语法： 

 G1 = {S->aA|bB, B->bB|b, A->aA|a} 
G2 = {S->aA|bB, B->bB|ε, A->aA|ε}

语法G1在GNF中，因为生产规则满足为GNF指定的规则。 但是，语法G2不在GNF中，因为生成规则B->ε和A->ε不满足为GNF指定的规则（只有起始符号可以生成ε）。
注意 -

  • 对于给定的语法，可以有多个GNF
  • GNF生成与CFG生成的语言相同的语言

如何将CFG转换为GNF -



步骤1.将语法转换为CNF。
如果给定的语法不在CNF中，则将其转换为CNF。 您可以参考以下文章将CFG转换为CNF： 将Context Free Grammar转换为Chomsky Normal Form

步骤2.如果存在，则从语法中消除左递归。
如果CFG包含左递归，则消除它们。 您可以参考以下文章来消除左递归： 解析| 第1组（简介，歧义和解析器）

步骤3.将生产规则转换为GNF形式。
如果任何生产规则不是GNF格式，请转换它们。 让我们举一个例子将CFG转换为GNF。 考虑给定的语法G1： 

 S → XA|BB 
B → b|SB 
X → b 
A → a

由于G1已经在CNF并且没有左递归，我们可以跳过步骤1和2并直接转到步骤3。
生产规则B-> SB不在GNF中，因此，我们将生产规则B-> SB中的S - > XA | BB替换为： 

 S → XA|BB 
B → b|XAB|BBB 
X → b 
A → a

生产规则S-> XA和B-> XAB不在GNF中，因此，我们将生产规则S-> XA和B-> XAB中的X-> b替换为： 

 S → bA|BB 
B → b|bAB|BBB 
X → b 
A → a

删除左递归（B-> BBB），我们得到： 

 S → bA|BB 
B → bC|bABC
C → BBC| ε
X → b 
A → a

删除null生产（C->ε），我们得到： 

 S → bA|BB 
B → bC|bABC|b|bAB
C → BBC|BB
X → b 
A → a

生产规则S-> BB不在GNF中，因此，我们将生产规则S-> BB中的B→bC | bABC | b | bAB替换为： 

 S → bA| bCB|bABCB|bB|bABB 
B → bC|bABC|b|bAB
C → BBC|BB
X → b 
A → a

生产规则C-> BB不在GNF中，因此，我们将生产规则C-> BB中的B→bC | bABC | b | bAB替换为： 

 S → bA| bCB|bABCB|bB|bABB 
B → bC|bABC|b|bAB
C → BBC
C → bCB|bABCB|bB|bABB
X → b 
A → a

生产规则C-> BBC不在GNF中，因此，我们将生产规则C-> BBC中的B→bC | bABC | b | bAB替换为： 

 S → bA| bCB|bABCB|bB|bABB 
B → bC|bABC|b|bAB
C → bCBC|bABCBC|bBC|bABBC
C → bCB|bABCB|bB|bABB
X → b 
A → a

这是语法G1的GNF格式。
