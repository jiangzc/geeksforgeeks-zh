有限自动机介绍

有限自动机（FA）是识别模式的最简单机器。

有限自动机包括以下内容： 

 Q : Finite set of states.
∑ : set of Input Symbols.
q : Initial state.
F : set of Final States.
δ : Transition Function.

机器的正式规格是
{Q，Σ，q，F，δ}。

FA有两种类型：


1）确定性有限自动机（DFA） 



 DFA consists of 5 tuples {Q, ∑, q, F, δ}. 
Q : set of all states.
∑ : set of input symbols. ( Symbols which machine takes as input )
q : Initial state. ( Starting state of a machine )
F : set of final state.
δ : Transition Function, defined as δ : QX ∑ --> Q.

在DFA中，对于特定输入字符，机器仅进入一个状态。 每个输入符号的每个状态都定义了一个转换函数。 同样在DFA null（或ε）中，移动是不允许的，即，DFA不能在没有任何输入字符的情况下改变状态。

例如，在具有Σ= {0,1}的DFA下面接受以0结尾的所有字符串。

需要注意的一件重要事情是， 模式可能有许多可能的DFA 。 通常优选具有最小状态数的DFA。


2）非确定性有限自动机（NFA）
除了以下附加功能外，NFA与DFA类似：
1.允许空（或ε）移动，即它可以向前移动而不读取符号。
2.能够转移到任何数量的状态以进行特定输入。
但是，上述功能不会为NFA增加任何功能。 如果我们在功率方面进行比较，两者都是等价的。

由于上述附加功能，NFA具有不同的过渡功能，其余部分与DFA相同。

一些要点：
1.每个DFA都是NFA，但反之亦然。
2. NFA和DFA都具有相同的权力，每个NFA都可以转换为DFA。
3. DFA和NFA中可以有多个最终状态。
3. NFA更像是一个理论概念。
4. DFA用于编译器中的词法分析。
