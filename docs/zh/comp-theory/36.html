构造语言L = {ww r |的图灵机 w∈{0,1}}

先决条件 - 图灵机
语言L = {ww r | w∈{0,1}}代表一种语言，你只使用2个字符，即0和1.语言的第一部分可以是0和1的任何字符串。第二部分是第一部分的反向。 将这两个部分组合成串。 属于此类别的任何此类字符串都将被此语言接受。 字符串的开头和结尾用$符号标记。

例如，如果第一部分w = 1 1 0 0 1然后第二部分w r = 1 0 0 1 1.可以清楚地看到w r是w的反向，所以字符串1 1 0 0 1 1 0 0 1 1是给定语言的一部分。

例子 - 

 Input : 0 0 1 1 1 1 0 0
Output : Accepted
Input : 1 0 1 0 0 1 0 1 
Output : Accepted

使用的方法 -
首先检查第一个符号，如果它是0，则将其替换为Y，如果是1则替换为X.然后转到字符串的结尾。 所以最后的符号与第一个符号相同。 我们也根据它将它替换为X或Y.
现在再次回到起始符号替换旁边的位置并重复上述相同的过程。
