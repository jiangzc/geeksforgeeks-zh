异常
在引用和引用关系中可能出现不同类型的异常，可以讨论如下：

插入异常：如果在引用关系中插入元组并且引用属性中不存在引用属性值，则它将不允许插入引用关系。 例如，如果我们尝试使用STUD_NO = 7在STUDENT_COURSE中插入记录，则不允许。

删除和更新异常：如果从引用的关系中删除或更新元组，并且通过引用关系中的引用属性使用引用的属性值，则不允许从引用的关系中删除元组。 例如，如果我们尝试使用STUD_NO = 1从STUDENT中删除记录，则不允许。 为避免这种情况，可以在查询中使用以下内容：

  • ON DELETE / UPDATE SET NULL：如果从引用关系中删除或更新元组，并且引用关系中的引用属性使用引用的属性值，它将从引用的关系中删除/更新元组，并将referenced属性的值设置为NULL。
  • ON DELETE / UPDATE CASCADE：如果从引用关系中删除或更新元组，并且通过引用关系中的引用属性使用引用的属性值，它将从引用的关系中删除/更新元组并引用关系。
