1）考虑银行账户上的两笔交易的以下日志序列，初始余额为12000，将2000转移到抵押付款，然后申请5％的利息。 

 1. T1 start 
   2. T1 B old=12000 new=10000 
   3. T1 M old=0 new=2000 
   4. T1 commit 
   5. T2 start 
   6. T2 B old=10000 new=10500 
   7. T2 commit

假设数据库系统在写入日志记录7之前崩溃。 重新启动系统时，哪一个语句对恢复过程是正确的？


（A）我们必须重做日志记录6以将B设置为10500
（B）我们必须撤消日志记录6以将B设置为10000，然后重做日志记录2和3
（C）我们不需要重做日志记录2和3，因为事务T1已经提交
（D）我们可以按任意顺序应用重做和撤消操作，因为它们是幂等的。

答案（B）



2）考虑注册的关系（学生，课程），其中（学生，课程）是主键，以及学生是主键的支付关系（学生，金额）。 假设没有空值，也没有外键或完整性约束。 鉴于以下四个问题： 

 Query1: select student from enrolled where student in (select student from paid)
Query2: select student from paid where student in (select student from enrolled)
Query3: select E.student from enrolled E, paid P where E.student = P.student
Query4: select student from paid where exists
        (select * from enrolled where enrolled.student = paid.student)

以下哪一项陈述是正确的？
（A）所有查询都返回任何数据库的相同行集
（B）Query2和Query4为所有数据库返回相同的行集，但存在Query1和Query2返回不同行集的数据库。
（C）存在Query3返回严格少于Query2的行的数据库。
（D）存在Query4在运行时将遇到完整性违规的数据库。



答案（B）
举个例子： 

 Table enrolled
student course
----------------
 abc c1   
 xyz c1
 abc c2
 pqr c1

Table paid
student amount
-----------------
 abc 20000
 xyz 10000
 rst 10000


Output of Query 1
 abc
 abc
 xyz

Output of Query 2
 abc
 xyz

Output of Query 3
 abc
 xyz

Output of Query 4
 abc
 xyz

查询1和查询3可以返回重复的学生值，因为“学生”不是注册关系中的关键字，但是查询2和查询4总是返回相同的行集。

因此，选项（B）是正确的。



3）考虑登记的关系（学生，课程），其中（学生，课程）是主要关键，以及支付的关系（学生，金额），其中学生是主要关键。 假设没有空值，也没有外键或完整性约束。 假设20％的学生每人支付6000,7000,8000,9000和10000的金额。 考虑这些查询计划（左侧的计划1，右侧的计划2）“列出已支付超过x的学生所采取的所有课程”。


磁盘搜索需要4毫秒，磁盘数据传输带宽为300 MB /秒，并检查元组以查看数量是否大于x需要10微秒。 以下哪项陈述是正确的？
（A）计划1和计划2不会为所有数据库输出相同的行集。
（B）对于某些数据库，计划1的输出中可能会多次列出一个课程
（C）对于x = 5000，对于所有数据库，计划1比计划2执行得更快。
（D）对于x = 9000，对于所有数据库，计划I执行的速度比计划2慢。

答案（C）
假设有足够大的内存可用于所需的所有数据。 两个计划都需要加载两个表课程并注册。 因此两个计划的磁盘访问时间都相同。
与计划1相比，计划2进行的比较次数较少。
1）连接操作将需要更多比较，因为与计划1相比，第二个表在计划2中将具有更多行。
2）两个表的连接表将有更多行，因此需要更多的比较来查找大于x的数量。



4）给出以下功能依赖性： 

 AB → CD, AF → D, DE → F, C → G , F → E, G → A

以下哪个选项是错误的？
（A）CF + = {ACDEFG}
（B）BG + = {ABCDG}
（C）AF + = {ACDEFG}
（D）AB + = {ABCDFG}

答案（C）
关闭AF或AF + = {ADEF}，关闭AF不包含C和G.
选项（D）看起来也是正确的。 AB + = {ABCDG}，AB的关闭不包含F.
