1）设r是模式R =（A，B，C，D）的关系实例。 我们定义r1 ='从r'选择A，B，C并且r2 ='从r'选择A，D。 设s = r1 * r2，其中*表示自然连接。 鉴于r分解为r1和r2是有损的，以下哪一个是真的？

3）考虑以下函数依赖性所持有的关系式R =（A，B，C，D，E，H）：{A-> B，BC-> D，E-> C，D->一个}。 R的候选键是什么？
（a）AE，BE
（b）AE，BE，DE
（c）AEH，BEH，BCH
（d）AEH，BEH，DEH

回答（d）
如果S的闭包是R的所有属性并且没有S的子集，其闭包是R的所有属性，则一组属性S是关系R的候选关键字。
关闭AEH，即AEH + = {ABCDEH}
关闭BEH，即BEH + = {ABCDEH}
关闭DEH，即DEH + = {ABCDEH}
