无损分解

如果以下函数依赖关系中的至少一个在F +中（关闭函数依赖关系），则将关系R分解为R1和R2是无损连接分解 

    
R1 ∩ R2 → R1
   OR
R1 ∩ R2 → R2

问题1：
设R（A，B，C，D）是具有以下函数依赖关系的关系模式： 

 A → B, B → C,
C → D and D → B. 

The decomposition of R into 
(A, B), (B, C), (B, D)

（A）给出无损连接，并且是依赖保持
（B）给出无损连接，但不是依赖保留
（C）不提供无损连接，但是依赖保留
（D）不提供无损连接，也不是依赖保留

请参阅此解决方案。
