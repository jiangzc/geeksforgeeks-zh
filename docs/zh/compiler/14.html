首先在语法分析中设置

语法符号X的FIRST（X）是开始可从X导出的字符串的终端集合。

计算FIRST集的规则：

  1. 如果x是终端，那么FIRST（x）= {'x'}
  2. 如果x->Є是生产规则，则将add添加到FIRST（x）。
  3. 如果X-> Y1 Y2 Y3 ... .Yn是生产，
    1. 第一（X）=第一（Y1）
    2. 如果FIRST（Y1）包含Є则FIRST（X）= {FIRST（Y1） - Є} U {FIRST（Y2）}
    3. 如果FIRST（Yi）包含所有i = 1到n的Є，则将add添加到FIRST（X）。

例1： 

 Production Rules of Grammar
E -> TE'
E' -> +T E'|Є
T -> F T'
T' -> *F T' | Є
F -> (E) | id

FIRST sets
FIRST(E) = FIRST(T) = { ( , id }
FIRST(E') = { +, Є }
FIRST(T) = FIRST(F) = { ( , id }
FIRST(T') = { *, Є }
FIRST(F) = { ( , id }

例2： 

 Production Rules of Grammar
S -> ACB | Cbb | Ba
A -> da | BC
B -> g | Є
C -> h | Є

FIRST sets
FIRST(S) = FIRST(A) U FIRST(B) U FIRST(C)
         = { d, g, h, Є, b, a}
FIRST(A) = { d } U FIRST(B) = { d, g , h, Є }
FIRST(B) = { g , Є }
FIRST(C) = { h , Є }

笔记：

  1. 上面使用的语法是Context-Free Grammar（CFG）。 可以使用CFG指定大多数编程语言的语法。
  2. CFG的形式为A - > B，其中A是单个非终端，B可以是一组语法符号（即终端以及非终端）
