什么是Spooling究竟是什么？

SPOOL是在线同步外围操作的首字母缩写。 它是一种缓冲机制或暂时保存数据以供设备，程序或系统执行的过程。 数据被发送并存储在存储器或其他易失性存储器中，直到程序或计算机请求执行。

在计算机系统中，诸如打印机和打卡读卡器等外围设备（批处理）相对于系统其余部分的性能而言非常慢。 从系统获取输入和输出很快就被视为瓶颈。 这需要线轴。

假脱机的工作方式类似于典型的请求队列，其中累积来自多个源的数据，指令和进程以供稍后执行。 通常，它保留在计算机的物理内存，缓冲区或I / O设备特定的中断上。 假脱机以FIFO方式处理，即队列中的任何第一条指令都将被弹出并执行。

线轴的应用/实现：

1）最常见的可以在键盘打印机和鼠标等I / O设备中找到。 例如，在打印机中，发送到打印机的文档/文件首先存储在内存或打印机假脱机程序中。 打印机准备就绪后，它会从假脱机中取出数据并打印出来。



即使经历了几秒钟你的鼠标或键盘停止工作的情况？ 同时，我们通常会在屏幕上一次又一次地点击以检查其是否正常工作。 当它实际开始工作时，我们在挂起状态期间按下的内容和位置执行得非常快，因为所有指令都存储在相应设备的假脱机中。

2）批处理系统使用假脱机来维护准备运行的作业队列，一旦系统有资源处理它们就可以启动。

3）假脱机能够将一个作业的I / O操作与另一个作​​业的处理器操作重叠。 即多个进程可以将文档写入打印队列而无需等待并继续工作。

4）电子邮件：MTA（邮件传输代理）将电子邮件发送到临时存储区域，等待MA（邮件用户代理）接收

5）也可以用于生成横幅页面（这些是用于计算机打印的页面，以便将文档彼此分开，并通过用户名，帐号或用于拾取的箱子来识别例如打印请求的发起者。页面用于办公环境中，许多人共享少量可用资源。
