重要的Linux命令（leave，diff，cal，ncal，locate和ln）

Linux提供了一些重要的技巧。 这是一些重要的：

  1. 请假 - 当你不得不离开时提醒你
    句法： 

     leave +hhmm

    请等到指定的时间（在接下来的12个小时内），然后提醒您必须通过写信给您执行的TTY离开。 提醒您在实际时间之前5分钟和1分钟，此时以及之后的每一分钟。
    选项： hhmm一天中的时间格式为hhmm，其中hh是以小时为单位的时间（12或24小时制），mm是分钟。

  2. diff - 逐行比较文件
    句法： 

     diff file1 file2

    逐行比较FILES。 



     diff -q file1 file2

    仅在文件不同时报告

  3. cal，ncal - 显示日历和复活节的日期
    句法： 

     cal

    cal实用程序以传统格式显示简单日历，ncal提供另一种布局，更多选项和复活节日期。 新的格式有点狭窄，但它适合25×80终端一年。 如果未指定参数，则显示当前月份。


  4. locate - 按名称查找文件
    句法： 

     locate file_name

    locate读取由updatedb（8）准备的一个或多个数据库，并将与至少一个PATTERN匹配的文件名写入标准输出，每行一个。

  5. passwd - 更改用户密码
    句法： 

     passwd

    passwd命令更改用户帐户的密码。 普通用户只能更改自己帐户的密码，而超级用户可以更改任何帐户的密码。 passwd还会更改帐户或相关的密码有效期。

  6. ln - 在文件之间建立链接
    句法： 

     ln existing_file_name file2_name

    使用指定的名称创建指向TARGET的链接
