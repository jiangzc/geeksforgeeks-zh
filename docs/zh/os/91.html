内存管理中的Next Fit算法程序

先决条件： 分区分配方法
什么是下一个适合？
下一个适合是'first fit'的修改版本。 它首先适合找到一个免费分区，但下次调用它时会从它停止的地方开始搜索，而不是从头开始搜索。 此策略使用流动指针。 指针沿着内存链漫游以搜索下一个拟合。 这有助于避免始终从空闲块链的头部（开始）使用内存。

与首次合身相比，它有什么优势？

  • 第一次拟合是一种直接快速的算法，但往往会将大部分自由部分切割成小块，因此，即使所有小块的总和大于所需要的，需要大部分内存块的进程也不会得到任何东西。这就是所谓的外部碎片问题。
  • 第一次拟合的另一个问题是它倾向于在存储器的开始处分配存储器部分，这可能在开始时导致更多的内部情况。 接下来试图通过开始搜索不是从内存开始的部分的空闲部分，而是从上次结束的位置开始搜索来解决这个问题。
  • 下一个拟合是一种非常快速的搜索算法，并且比First Fit和Best Fit Memory Management算法相对更快。 
 Example:
Input : blockSize[] = {5, 10, 20};
     processSize[] = {10, 20, 30};
Output:
Process No. Process Size Block no.
 1 10 2
 2 20 3
 3 30 Not Allocated

建议：在继续使用解决方案之前，请先在{IDE}上尝试您的方法。



算法：

  1. 输入内存块的数量及其大小，并将所有块初始化为空闲。
  2. 输入流程数量及其大小。
  3. 首先选择每个进程并检查它是否可以分配给当前块，如果是，则为其分配所需的内存并检查下一个进程，但是从我们离开的块开始。
  4. 如果当前块大小较小，则继续检查其他块。 

// C/C++ program for next fit
// memory management algorithm
#include <bits/stdc++.h>
using namespace std;

// Function to allocate memory to blocks as per Next fit
// algorithm
void NextFit(int blockSize[], int m, int processSize[], int n)
{
    // Stores block id of the block allocated to a
    // process
    int allocation[n], j = 0;

    // Initially no block is assigned to any process
    memset(allocation, -1, sizeof(allocation));

    // pick each process and find suitable blocks
    // according to its size ad assign to it
    for (int i = 0; i < n; i++) {

        // Do not start from beginning
        while (j < m) {

            if (blockSize[j] >= processSize[i]) {

                // allocate block j to p[i] process
                allocation[i] = j;

                // Reduce available memory in this block.
                blockSize[j] -= processSize[i];

                break;
            }

            // mod m will help in traversing the blocks from
            // starting block after we reach the end. 
            j = (j + 1) % m;
        }
    }

    cout << "\nProcess No.\tProcess Size\tBlock no.\n";
    for (int i = 0; i < n; i++) {
        cout << " " << i + 1 << "\t\t" << processSize[i] 
             << "\t\t";
        if (allocation[i] != -1)
            cout << allocation[i] + 1;
        else
            cout << "Not Allocated";
        cout << endl;
    }
}

// Driver program
int main()
{
    int blockSize[] = { 5, 10, 20 };
    int processSize[] = { 10, 20, 5 };
    int m = sizeof(blockSize) / sizeof(blockSize[0]);
    int n = sizeof(processSize) / sizeof(processSize[0]);

    NextFit(blockSize, m, processSize, n);

    return 0;
}
