多路复用器| 数字电子

它是一个组合电路，根据控制或选择输入有许多数据输入和单输出。对于N条输入线，log n（base2）选择线，或者我们可以说对于2 n条输入线，需要n条选择线。 多路复用器也称为“数据n选择器，与串行转换器并行，多对一电路，通用逻辑电路” 。 多路复用器主要用于增加可在一定时间和带宽内通过网络发送的数据量。

通常，为了使用A：1 MUX实现B：1 MUX，使用一个公式来实现它。
B / A = K1，
K1 / A = K2，
K2 / A = K3

..................

K N-1 / A = K N = 1（直到我们获得1个MUX计数）。

然后添加MUX的所有数量= K1 + K2 + K3 + .... + K N.
例如：使用4：1 MUX实现64：1 MUX
使用上面的公式，我们可以获得相同的。
64/4 = 16
16/4 = 4
4/4 = 1（直到我们获得1个MUX计数）
因此，实现64：1 MUX = 16 + 4 + 1 = 21需要4：1 MUX的总数。
