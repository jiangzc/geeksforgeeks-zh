功能完整性

当且仅当每个切换功能可以通过其中的操作表达时，一组操作被认为是功能完整的或通用的。 如果可以从该集合构造所有其他布尔函数并且提供一组输入变量，则一组布尔函数在功能上是完整的，例如，

  • 设置A = {+，*，'（OR，AND，补码）}在功能上是完整的。
  • 设置B = {+，'}在功能上是完整的
  • 集合C = {*，'}在功能上是完整的

Post的功能完备性定理 -重要的封闭功能类：

  1. T 0 - 所有0保留函数的类，例如f（0,0，...，0）= 0。
  2. T 1 - 所有1保留函数的类，例如f（1,1，...，1）= 1。
  3. S - 类自对偶函数，例如f（x 1 ，...，x n ）=¬f（¬x1，...，¬xn）。
  4. M - 类单调函数，如：{x 1 ，...，x n }≤{x 1 ，...，x n }，如果xi≤yi
    如果{x 1 ，...，x n }≤{x 1 ，...，x n }
    那么f（x 1 ，...，x n ）≤f（x 1 ，...，x n ）
  5. L - 类线性函数，可表示为：f（x 1 ，...，x n ）= a 0 + a 1 ·x 1 + ... + a n ·x n ; a i {0,1}。

定理 -布尔函数系统在功能上是完整的，当且仅当对于五个定义的类T 0 ，T 1 ，S，M，L中的每一个，存在不属于该类的F的成员。

功能完整性的例子 -

  • 检查功能F（A，B，C）= A'+ BC'功能是否完整？
  • 解释 -让我们从把所有变量都放到'A'开始吧
    F（A，A，A）= A'+ A.A'= A' - （i）
    F（B，B，B）= B'+ B.B'= B' - （ii）
    现在用F（A，A，A）代替变量'A'和F（B，B，B）代替变量'C'
    F（F（A，A，A），B，F（B，B，B））=（A'）'+ B.（B'）'= A + B-（iii）
    从（i）和（ii）得到补，并且从（iii）得到运算符'+'，因此如果函数包含{+，'}在功能上完成 ，则该函数在功能上完成 。
  • 检查功能F（A，B）= A'+ B功能是否完整？
  • 解释 -让我们从把所有变量都放到'A'开始吧
    F（A，A）= A'+ A'= 1 - （i）
    F（B，B）= B'+ B'= 1-（ii）
    F（A，0）= A'+ 0 = A' - （iv）
    现在用F（A，0）代替变量'A'
    F（F（A，0），B）=（A'）'+ B = A + B-（iii）
    从（iv）得到补，从（iii）得到运算符'+'，因此如果函数包含{+，'} 部分功能完整 ，则此函数在功能上完成 。
  • 检查功能F（A，B）= A'B功能是否完整？
  • 解释 -让我们从把所有变量都放到'A'开始吧
    F（A，A）= A'.A'= 0 - （i）
    F（A，0）= A'.0 = 0-（ii）
    F（A，1）= A'.1 = A' - （iv）
    现在用F（A，1）代替变量'A'
    F（F（A，1），B）=（A'）'* B = A * B-（iii）
    从（iv）得到补，并且从（iii）得到运算符'*'，因此如果函数包含{*，'} 部分功能完整 ，则该函数在功能上完成 。

    注 -如果函数通过替换“0”或“1”在功能上完成，那么它被称为部分功能完整。

  • 检查功能F（A，B）= A'B + AB'（EX-OR）功能是否完整？
  • 解释 -让我们从把所有变量都放到'A'开始吧
    F（A，1）= A'.1 + A.0 = A' - （i）
    F（A'，B）= AB + A'B' - （ii）
    F（A'，B'）= AB'+ A'B-（iii）
    F（A，B'）= A'B'= AB-（iv）
    因此根据条件无法获得{+，*，'}。 所以EX-OR在非功能上是完整的 。
