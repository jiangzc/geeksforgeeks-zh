共识定理

先决条件 - 布尔代数的性质 ，布尔函数的最小化

冗余定理用作数字电子中的布尔代数技巧。 它也被称为共识定理： 

 AB + A'C + BC = AB + A'C

AB和A'C术语的共识或解决方案是BC。 它是术语所有独特文字的结合，不包括在一个术语中看起来不必要而在另一个术语中被否定的文字。

这个等式的联合对偶是： 

 (A+B).(A'+C).(B+C) = (A+B).(A'+C)

在第二行中，我们省略了第三个产品术语BC.Here，术语BC称为冗余术语。 通过这种方式，我们将这个定理简单地用于布尔代数。 应用冗余定理的条件是：



  1. 表达式中必须存在三个变量。这里使用A，B和C作为变量。
  2. 每个变量重复两次。
  3. 一个变量必须以补充形式出现。

应用此定理后，我们只能采用包含补充变量的那些术语。

证明 -我们也可以证明这样： 

 Y = AB + A'C + BC
Y = AB + A'C + BC.1
Y = AB + A'C + BC.(A + A')
Y = AB + A'C + ABC + A'BC
Y = AB(1 + C) + A'C(1 + B)
Y = AB + A'C

实施例-1。 

 F = AB + BC' + AC

这里，我们有三个变量A，B和C，并且都重复两次。 变量C以补充形式存在。 因此，应用该定理满足所有条件。

在应用冗余定理之后，我们只能写出包含补充变量（即C）的项，并省略冗余项，即AB。 

 .'. F = BC' + AC

例2。 

 F = (A + B).(A' + C).(B + C)

存在三个变量，并且所有变量都重复两次。 变量A以互补的形式存在。因此，满足该定理的所有三个条件。

在应用冗余定理之后，我们只能写出包含补充变量的项（即A），并省略冗余项，即（B + C）。 

 .'. F = (A + B).(A' + C)

考虑以下等式： 

 Y = AB + A'C + BC

第三个产品术语BC是一个冗余的共识术语。 如果A从1切换到0而B = 1且C = 1，则Y保持为1.在逻辑门中信号A的转换期间，第一和第二项都可以暂时为0。 第三项可防止毛刺，因为在这种情况下其值1不受信号A转换的影响。

从而。 删除逻辑冗余非常重要，因为它会导致不必要的网络复杂性并增加实施成本。

因此，通过这种方式，我们可以最小化布尔表达式来解决它。
