移位寄存器

触发器可用于存储单个位的二进制数据（1或0）。 然而，为了存储多位数据，我们需要多个触发器。 N个触发器按顺序连接以存储n位数据。 寄存器是用于存储此类信息的设备。 它是一组串联连接的触发器，用于存储多位数据。

存储在这些寄存器中的信息可以在移位寄存器的帮助下传输。 移位寄存器是一组用于存储多位数据的触发器。 通过施加时钟脉冲，可以使存储在这些寄存器中的位在寄存器内移动/移出/移出寄存器。 可以通过连接n个触发器来形成n位移位寄存器，其中每个触发器存储单个数据位。
将位移到左侧的寄存器称为“左移位寄存器”。
将位移位到右侧的寄存器称为“右移位寄存器”。

移位寄存器基本上有4种类型。 这些是：

  1. 串行输入串行输出移位寄存器
  2. 串行并行输出移位寄存器
  3. 并行输入串行输出移位寄存器
  4. 并行输出移位寄存器

串行输出串行移位寄存器（SISO） -

移位寄存器允许串行输入（通过单条数据线一位接一位）并产生串行输出，称为串行输出串行移位寄存器。 由于只有一个输出，因此数据以串行模式一次一位地离开移位寄存器，因此称为串行输出串行移位寄存器。

串并输出移位寄存器（SIPO） -

移位寄存器允许串行输入（通过单条数据线一位接一位）并产生并行输出，称为串行并行输出移位寄存器。

上述电路是右移寄存器的一个例子，从触发器的左侧输入串行数据并产生并行输出。 它们用于通信线路，其中需要将数据线解复用成多个并行线路，因为SIPO寄存器的主要用途是将串行数据转换成并行数据。

并行输入移位寄存器（PISO） -

移位寄存器允许并行输入（数据分别给予每个触发器并同时进行）并产生串行输出，称为并行输出串行移位寄存器。

下面给出的逻辑电路显示并行输入串行输出移位寄存器。 该电路由四个连接的D触发器组成。 时钟输入直接连接到所有触发器，但输入数据在每个触发器的输入端通过多路复用器单独连接到每个触发器。 前一个触发器和并行数据输入的输出连接到MUX的输入，MUX的输出连接到下一个触发器。 所有这些触发器彼此同步，因为相同的时钟信号被施加到每个触发器。

并行输出（PISO）移位寄存器用于将并行数据转换为串行数据。

并行输出并行移位寄存器（PIPO） -

移位寄存器允许并行输入（数据分别给予每个触发器并同时进行）并产生并行输出，称为并行输出并行输出移位寄存器。

下面给出的逻辑电路显示并行输出并行输出寄存器。 该电路由四个连接的D触发器组成。 清零（CLR）信号和时钟信号连接到所有4个触发器。 在这种类型的寄存器中，各个触发器之间没有互连，因为不需要数据的串行移位。 对于每个触发器，数据作为输入单独给出，并且以相同的方式，也从每个触发器单独收集输出。

并行输出并行输出（PIPO）移位寄存器用作临时存储设备，与SISO移位寄存器一样，它用作延迟元件。

双向移位寄存器 -

如果我们将二进制数向左移动一个位置，则相当于将数字乘以2，如果我们将二进制数向右移动一个位置，则相当于将数字除以2.执行这些操作我们需要一个可以向任一方向移动数据的寄存器。

双向移位寄存器是能够根据所选模式向右或向左移位数据的寄存器。 如果所选模式为1（高），则数据将向右移动，如果所选模式为0（低），则数据将向左移动。

移位寄存器的应用 -

  • 移位寄存器用于临时数据存储。
  • 移位寄存器也用于数据传输和数据操作。
  • 串行输入串行输出和并行输出并行输出寄存器用于产生数字电路的时间延迟。
  • 串行输出并行输出移位寄存器用于将串行数据转换为并行数据，因此它们用于需要将数据线解复用成多个并行线的通信线路中。
  • 并行输出移位寄存器用于将并行数据转换为串行数据。
