运算放大器（op-amp）

放大器是一种增加输入信号强度的器件。 它可以是电压放大器，其输入是一些电压，输出也是电压但放大。 电流放大器，其输入是一些电流和输出也是电流但放大。

  • 跨导放大器 ，其输入是一些电压，输出是电流。
  • 互阻抗放大器 ，其输入为电流，输出为电压。

运算放大器（或运算放大器）是一种电压放大，三端子电子设备，具有两个输入端子，即反相端子（标有' - '符号图）和非反相端子（标有'+'符号图） ）和第三个终端是输出终端。 运算放大器的增益（“A”）=输出信号/输入信号

运算放大器的不同配置：

开环配置 -
在这种配置中，运算放大器没有任何反馈。理想情况下，它具有无限的开环增益（几乎是其输入端之间电位差的数十万倍）。

理想运算放大器的特性 -

  • 开环增益：理想情况下，运算放大器应具有无限的开环增益（实际上它比输入端之间的电位差大几十万倍）。
  • 输入阻抗或电阻：理想情况下，运算放大器应具有无限输入电阻（实际上它应该非常高）。
  • 输出阻抗或电阻：理想情况下，运算放大器应具有零输出电阻（实际上它应该非常低）。
  • 带宽：理想情况下，运算放大器应具有无限带宽（实际上它是有限的）。

基本术语 -

  1. 输出失调电压：当输入之间的电压差为零时，运算放大器的输出应该理想地为零，但实际上输出非零，有一个非常小的电压。当没有时，输出侧的这个不需要的电压输入被称为输出偏移电压。
  2. 输入失调电流：当运算放大器没有输入电压时，进入反相和非反相端子的电流差值的大小。 
     Io = |Ib1-Ib2|; 
Io-Input Offset Current, Ib1 & 
Ib2-current at input terminals
  3. 输入偏置电流： 
     I(bias) = (Ib1+Ib2)/2
  4. 输入失调电压：它是在运算放大器的反相或非反相端故意施加的电压，以抵消输出失调电压的影响。 
     V(Input Offset Voltage) = 0 (ideally)
V(Input Offset Voltage) = -V(Output Offset Voltage) (practically)
  5. 共模抑制比：它是差模增益（当不同信号施加到两个输入端子时）与共模增益之间的比率（当信号仅施加到输入端子之一时）。 
     e(Common Mode Rejection Ratio) = |(differential mode gain) / (common mode gain)|

应用 -它可以用作：

  • 反相和非反相加法器，
  • 减法器，
  • 系统集成商，
  • 微分，
  • 对数放大器等
