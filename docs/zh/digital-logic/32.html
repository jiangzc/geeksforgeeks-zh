幅度比较器

幅度数字比较器是一种组合电路，它比较两个数字或二进制数 ，以便找出一个二进制数是否等于，小于或大于另一个二进制数。 我们逻辑上设计了一个电路，我们将有两个输入，一个用于A，另一个用于B，有三个输出端子，一个用于A> B条件，一个用于A = B条件，一个用于A <B条件。

1位幅度比较器 -

用于比较两个比特的比较器称为单比特比较器。 它由两个输入组成，每个输入用于两个单个位数，三个输出用于生成小于，等于和大于两个二进制数之间的输入。

4位幅度比较器 -

用于比较两个二进制数的比较器，每个四位被称为4位幅度比较器。 它由8个输入组成，每个输入用于两个四位数和三个输出，以产生小于，等于和大于两个二进制数之间的值。

在4位比较器中，在以下四种情况下可以实现A> B的条件：

  1. 如果A3 = 1且B3 = 0
  2. 如果A3 = B3且A2 = 1且B2 = 0
  3. 如果A3 = B3，则A2 = B2且A1 = 1且B1 = 0
  4. 如果A3 = B3，A2 = B2，A1 = B1且A0 = 1且B0 = 0

类似地，在以下四种情况下，A <B的条件是可能的：

  1. 如果A3 = 0且B3 = 1
  2. 如果A3 = B3且A2 = 0且B2 = 1
  3. 如果A3 = B3，则A2 = B2且A1 = 0且B1 = 1
  4. 如果A3 = B3，A2 = B2，A1 = B1，A0 = 0，B0 = 1

只有当一个数字的所有单个位与另一个数字的相应位完全一致时，A = B的条件才是可能的。

级联比较器 -

通过级联两个或多个4位比较器来执行比较操作超过四位的比较器称为级联比较器。 当两个比较器要级联时，低阶比较器的输出连接到高阶比较器的相应输入。

比较器的应用 -

  1. 比较器用于中央处理单元（CPU）和微控制器（MCU）。
  2. 这些用于控制应用，其中表示诸如温度，位置等物理变量的二进制数与参考值进行比较。
  3. 比较器还用作过程控制器和伺服电机控制。
  4. 用于密码验证和生物识别应用程序。
