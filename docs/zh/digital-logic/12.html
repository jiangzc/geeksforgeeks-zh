规范和标准表格

规范形式 -在布尔代数中，布尔函数可以表示为Canonical Disjunctive Normal Form，称为minterm ，有些则表示为Canonical Conjunctive Normal Form，称为maxterm 。
在Minterm中，我们寻找输出结果为“1”的函数，而在Minterm中我们寻找输出结果为“0”的函数。
我们执行minterm的总和，也称为产品总和（SOP）。
我们执行Maxterm的产品，也称为sum（POS）产品。
表示为minterms之和或maxterms乘积的布尔函数被称为规范形式。

标准形式 -布尔变量可以用真实形式或补充形式表示。 在标准形式中，布尔函数将包含真实形式或补充形式的所有变量，而规范数量的变量取决于SOP或POS的输出。

布尔函数可以通过形成以下公式从给定的真值表中以代数方式表示：

  • minterm为每个在函数中产生1的变量组合，然后取所有这些项的OR。
  • maxterm为函数中产生0的每个变量组合，然后取所有这些项的AND。

minterms的总和 -
总和定义布尔函数的minterms是在真值表中给出函数的1的那些。 由于每个minterm的函数可以是1或0，并且因为有2 ^ n个minterms，所以可以计算所有可以用n个变量形成的函数（2 ^（2 ^ n））。 有时在minterm形式的总和中表达布尔函数是很方便的。

  • 示例 -将布尔函数F = A + B'C表示为minterms的总和。
  • 方案 -
    A = A（B + B'）= AB + AB'
    这个函数仍然缺少一个变量，所以
    A = AB（C + C'）+ AB'（C + C'）= ABC + ABC'+ AB'C + AB'C'
    第二个词B'C缺少一个变量; 因此，
    B'C = B'C（A + A'）= AB'C + A'B'C
    结合所有条款，我们有
    F = A + B'C = ABC + ABC'+ AB'C + AB'C'+ A'B'
    但AB'C出现两次，而且
    根据定理1（x + x = x），可以去除其中一个出现。 重新排列minterms，我们终于获得了
    F = A'B'C + AB'C + AB'C + ABC'+ ABC
    = m1 + m4 + m5 + m6 + m7
    SOP表示为Sigma（1,4,5,6,7）

      • maxterms的产品 -
      n个二进制变量的2 ^ 2n个函数中的每一个也可以表示为maxterms的乘积。 要将布尔函数表示为maxterms的乘积，必须首先将其带入OR项的形式。

      • 示例 -将布尔函数F = xy + x'z表示为maxterms的乘积
      • 方案 -
        F = xy + x'z
        =（xy + x'）（xy + z）
        =（x + x'）（y + x'）（x + z）（y + z）
        =（x'+ y）（x + z）（y + z）
        x'+ y = x'+ y + zz'
        =（x'+ y + z）（x'+ y + z'）x + z
        = x + z + yy'
        =（x + y + z）（x + y'+ z）y + z
        = y + z + xx'
        =（x + y + z）（x'+ y + z）
        F =（x + y + z）（x + y'+ z）（x'+ y + z）（x'+ y + z'）
        = M0 * M2 * M4 * M5
        POS表示为Pi（0,2,4,5）

        • 规范表格之间的转换 -
        表示为minterms之和的函数的补码等于原始函数中缺失的minterms的总和。 这是因为原始函数由使函数等于1的那些minterms表示，而对于函数为0的那些minterms，它的补码是1。

        • 示例 -

          F（A，B，C）=西格玛（1,4,5,6,7）
          F'（A，B，C）= Sigma（0,2,3）= m0 + m2 + m3
          现在，如果我们用DeMorgan定理得到F'的补充，我们得到一个不同形式的F：
          F =（m0 + m2 + m3）'
          = m0'm2'm3'
          = M0 * M2 * M3
          = PI（0,2,3）

        标准格式转换 -
        布尔代数的两种规范形式是通过从真值表中读取给定函数而获得的基本形式。 这些形式很少是文字数量最少的形式，因为根据定义，每个minterm或maxterm必须包含所有变量，无论是补充的还是未实现的。

        • 示例 -以标准形式F = y'+ xz'+ xyz转换布尔表达式
        • 解 - F =（x + x'）y'（z + z'）+ x（y + y'）z'+ xyz
          F = xy'z + xy'z'+ x'y'z + x'y'z'+ xyz'+ xy'z'+ xyz
