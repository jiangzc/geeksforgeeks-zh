K-Map（卡诺图）


在许多数字电路和实际问题中，我们需要找到具有最小变量的表达式。 我们可以使用K-map非常容易地最小化3,4个变量的布尔表达式，而无需使用任何布尔代数定理。 K-map可以根据问题的需要采用两种形式的产品总和（SOP）和总和产品（POS）。 K-map是表格式表示，但它提供的信息比TRUTH TABLE更多。 我们用0和1填充K-map的网格，然后通过制作组来解决它。

使用K-map解决表达的步骤 -

  1. 根据变量的数量选择K-map。
  2. 识别问题中给出的minterms或maxterms。
  3. 对于SOP，将1个放在K-map的块中，分别对应于minterms（0在其他地方）。
  4. 对于POS，将0放在K-map的块中，分别对应于maxterms（其他地方为1）。
  5. 制作矩形组，其中包含2个幂的总项，如2,4,8 ..（1除外），并尝试在一组中覆盖尽可能多的元素。
  6. 从步骤5中制作的组中查找产品术语并将其总结为SOP表单。

F（P，Q，R，S）=Σ（0,2,5,7,8,10,13,15）

从红色组我们得到产品术语 -

QS

从绿色组我们得到产品术语 -

Q'S特快”

总结这些产品术语我们得到 - 最终表达式（QS + Q'S'）

POS表格

  1. 3个变量的K图 -

F（A，B，C）=π（0,3,6,7）

从红色组我们找到条款

A B C'

补充这两个

'B'C

现在总结一下

（A'+ B'+ C）

从绿色组我们找到条款

B C.

补充这两个术语

B'C'

现在总结一下

（B '+ C'）

从棕色组我们找到条款

一个'B'C'

补充这两个

ABC

现在总结一下

（A + B + C）

我们将采用这三个术语的乘积： 最终表达式（A'+ B'+ C）（B'+ C'）（A + B + C）

2. 4个变量的K图 -

F（A，B，C，d）=π（3,5,7,8,10,11,12,13）

从绿色组我们找到条款

C'D B.

把他们的补充和总结

（C + d '+ B'）

从红色组我们找到条款

C D A'

把他们的补充和总结

（C '+ d' + A）

从蓝色组我们找到条款

A C'D'

把他们的补充和总结

（A'+ C + d）

从棕色组我们找到条款

A B'C

把他们的补充和总结

（A '+ B + C'）

最后我们将这些表示为产品 - （C + D'+ B'）。（C'+ D'+ A）。（A'+ C + D）。（A'+ B + C'）
