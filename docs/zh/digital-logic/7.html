十进制到八进制转换的程序

给定十进制数作为输入，我们需要编写一个程序将给定的十进制数转换为等效的八进制数。 即将数字与基值10转换为基数值8.数字系统的基值确定用于表示数值的位数。 例如，二进制数系统使用两个数字0和1，八进制数系统使用0-7的8位数，十进制数系统使用10位数0-9表示任何数值。

例子： 

 Input : 16
Output : 20

Input : 10
Output : 12

Input: 33
Output: 41

建议：在继续使用解决方案之前，请先在{IDE}上尝试您的方法。



算法 ：

  1. 在数组中将数字除以8时存储余数。
  2. 现在将数字除以8
  3. 重复上述两个步骤，直到数字不等于0。
  4. 现在以相反的顺序打印阵列。

例如：
如果给定的十进制数是16。
步骤1 ：当16除以8时的余数为0.因此，arr [0] = 0。
第2步 ：将16除以8.新数是16/8 = 2。
步骤3 ：当2除以8时的余数为2.因此，arr [1] = 2。
步骤4 ：将2除以8.新数字是2/8 = 0。
步骤5 ：由于number变为= 0.停止重复步骤并以相反的顺序打印数组。 因此，等效八进制数为20。

C / C ++ 

// C++ program to convert a decimal
// number to octal number

#include <iostream>
using namespace std;

// function to convert decimal to octal
void decToOctal(int n)
{
    
    // array to store octal number
    int octalNum[100];

    // counter for octal number array
    int i = 0;
    while (n != 0) {

        // storing remainder in octal array
        octalNum[i] = n % 8;
        n = n / 8;
        i++;
    }

    // printing octal number array in reverse order
    for (int j = i - 1; j >= 0; j--)
        cout << octalNum[j];
}

// Driver program to test above function
int main()
{
    int n = 33;

    decToOctal(n);

    return 0;
}

// Java program to convert a decimal
// number to octal number
import java.io.*;

class GFG 
{
    // Function to convert decimal to octal
    static void decToOctal(int n)
    {
        // array to store octal number
        int[] octalNum = new int[100];

        // counter for octal number array
        int i = 0;
        while (n != 0) 
        {
            // storing remainder in octal array
            octalNum[i] = n % 8;
            n = n / 8;
            i++;
        }

        // Printing octal number array in reverse order
        for (int j = i - 1; j >= 0; j--)
            System.out.print(octalNum[j]);
    }
    
    // driver program
    public static void main (String[] args) 
    {
        int n = 33;
        decToOctal(n); 
    }
}
