n位约翰逊计数器

先决条件 - 计数器

约翰逊计数器也称为爬行计数器，是同步计数器的一个例子。 在约翰逊计数器中，最后一个触发器的补码输出连接到第一个触发器的输入，并实现n位约翰逊计数器，我们需要n个flip -flop。它是最重要的移位寄存器计数器之一。 它是由输出反馈到自己的输入而形成的。约翰逊计数器是一个带反转的环。约翰逊计数器的另一个名称是：爬行计数器，扭环计数器，行走计数器，移动计数器和开关尾计数器。

n位Johnson计数器中已使用和未使用状态的总数：
使用状态数= 2n
未使用状态数= 2 n - 2 * n

如果n = 4

一切都有一些优点和缺点。
Johnson专柜的优点：

  • Johnson计数器具有相同数量的触发器，但它可以计数环计数器可以计数的状态数的两倍。
  • 它可以使用D和JK触发器实现。
  • Johnson环计数器用于计算连续循环中的数据。
  • Johnson计数器是一种自解码电路。

约翰逊专柜的缺点：

  • 约翰逊计数器不计入二进制序列。
  • 在约翰逊的反击中，更多的州仍然没有被利用的国家数量。
  • 所需的触发器数量是定时信号数量的一半。
  • 它可以构造成任意数量的时序。

Johnson专柜的应用：

  • Johnson计数器用作同步十进制计数器或分频器电路。
  • 它用于硬件逻辑设计，以创建复杂的有限状态机。 例如：ASIC和FPGA设计。
  • 3级约翰逊计数器用作3相方波发生器，产生1200相移。
  • 它用于通过改变反馈来划分时钟信号的频率。
