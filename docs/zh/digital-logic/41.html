Master Slave JK Flip Flop

先决条件 - 触发器类型及其转换

在JK触发器周围环境中竞争 -对于JK触发器，如果​​J = K = 1，并且如果clk = 1很长一段时间，那么只要CLK为高电平，Q输出就会切换，这使得输出触发器不稳定或不确定。 这个问题在JK触发器中被称为竞争条件。 通过确保时钟输入仅在非常短的时间内处于逻辑“1”，可以避免该问题（Race Around Condition）。 这引入了Master Slave JK触发器的概念。

Master Slave JK触发器 -
主从触发器基本上是两个串联配置连接在一起的JK触发器的组合。 其中，一个充当“主人”而另一个充当“奴隶” 。 主触发器的输出连接到从触发器的两个输入，其输出被反馈到主触发器的输入。

除了这两个触发器之外，该电路还包括一个逆变器 。 反相器连接到时钟脉冲，使得反相时钟脉冲被提供给从触发器。 换句话说，如果主触发器的CP = 0，则对于从触发器CP = 1，对于主触发器，如果​​CP = 1，则对于从触发器它变为0。



使用主从触发器 -

  1. 当时钟脉冲变为1时，从机被隔离; J和K输入可能会影响系统的状态。 从触发器被隔离，直到CP变为0.当CP返回到0时，信息从主触发器传递到从器件并获得输出。
  2. 首先，主触发器是正电平触发，从触发器是负电平触发，因此主器件在从器件之前响应。
  3. 如果J = 0且K = 1，则主机的高Q'输出进入从机的K输入，时钟强制从机复位，因此从机复制主机。
  4. 如果J = 1且K = 0，则主机的高Q输出进入从机的J输入，时钟的负转换设置从机，复制主机。
  5. 如果J = 1且K = 1，则它在时钟的正跳变时切换，因此从机在时钟的负跳变上切换。
  6. 如果J = 0且K = 0，则触发器被禁用且Q保持不变。

主触发器的时序图 -

  1. 当时钟脉冲为高电平时，主机的输出为高电平，并保持高电平直到时钟为低电平，因为存储了状态。
  2. 现在，当时钟脉冲再次变为高电平时，主机的输出变为低电平，并且在时钟再次变为高电平之前保持低电平。
  3. 因此，切换发生在一个时钟周期。
  4. 当时钟脉冲为高电平时，主机工作但不是从机，因此从机的输出保持低电平，直到时钟保持高电平。
  5. 当时钟为低电平时，从器件开始工作并保持高电平，直到时钟再次变低。
  6. 切换发生在整个过程中，因为输出在一个循环中改变一次。

这使得主从JK触发器成为同步器件，因为它仅通过时钟信号的定时传递数据。
